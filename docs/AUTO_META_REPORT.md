# 메타광고 리포트 자동화 시스템

모든 비즈니스 포트폴리오에 대한 일일/주간 메타광고 성과 리포트를 자동으로 생성하고 노션에 업로드합니다.

---

## 📋 개요

### 일일 리포트
- **실행 시간**: 매일 오전 9시 (KST)
- **실행 조건**: 평일만 (토/일/공휴일 제외)
- **리포트 기간**: 전날 데이터
  - 주말/공휴일 다음날은 해당 기간만큼 리포트 (예: 월요일에 금/토/일 3일치)
- **자동 노션 업로드**: ✅

### 주간 리포트
- **실행 시간**: 매주 월요일 오전 9시 (KST)
- **리포트 기간**: 지난 7일간 데이터
- **공휴일 영향**: 없음 (공휴일 상관없이 7일간)
- **자동 노션 업로드**: ✅

## 📊 생성되는 리포트

각 클라이언트별로 다음 4개 파일이 생성됩니다:

1. **분석_리포트.md** - 상세 분석 리포트
   - 전체 성과 요약
   - 핵심 인사이트 3가지
   - 광고별 성과 분석
   - 즉시 실행 액션
   - 예상 개선 효과

2. **1페이지_요약.md** - 경영진 보고용 요약

3. **실행_체크리스트.md** - 실무자용 액션 아이템

4. **상세_데이터.csv** - 분석용 원본 데이터

## 🚀 실행 방법

### 로컬에서 테스트

#### 일일 리포트 테스트
```bash
python3 scripts/auto_daily_meta_report.py
```

#### 주간 리포트 테스트
```bash
python3 scripts/auto_weekly_meta_report.py
```

### GitHub Actions (자동)

워크플로우가 자동으로 실행됩니다:

- **일일**: `.github/workflows/daily-meta-report.yml`
  - 스케줄: `0 0 * * 1-5` (월~금 오전 9시 KST)

- **주간**: `.github/workflows/weekly-meta-report.yml`
  - 스케줄: `0 0 * * 1` (매주 월요일 오전 9시 KST)

### 수동 실행

GitHub Actions 페이지에서 "Run workflow" 버튼으로 수동 실행 가능:

1. GitHub 저장소 → Actions 탭
2. "Daily Meta Ad Report" 또는 "Weekly Meta Ad Report" 선택
3. "Run workflow" 클릭

## 🔧 설정

### 필수 환경 변수 (.env)

```bash
# Meta API
META_APP_ID=your_app_id
META_APP_SECRET=your_app_secret
META_ACCESS_TOKEN=your_access_token

# Notion
NOTION_TOKEN=your_notion_token
NOTION_PAGE_ID_MARKETING=your_page_id
```

### GitHub Secrets

GitHub 저장소 Settings → Secrets and variables → Actions에서 설정:

- `META_APP_ID`
- `META_APP_SECRET`
- `META_ACCESS_TOKEN`
- `NOTION_TOKEN`
- `NOTION_PAGE_ID_MARKETING`

## 🛠️ 트러블슈팅

### CSV 파일이 없다는 오류

**문제**: `⚠️ [클라이언트명] CSV 파일 없음`

**해결**:
1. `input/meta-ad/[클라이언트명]/` 폴더에 CSV 파일이 있는지 확인
2. Meta Marketing API를 사용한 자동 다운로드 구현 필요 (현재는 수동)

### 노션 업로드 실패

**문제**: `❌ [클라이언트명] 노션 업로드 실패`

**해결**:
1. `.env` 파일에 `NOTION_TOKEN`과 `NOTION_PAGE_ID_MARKETING`이 설정되어 있는지 확인
2. 노션 API 토큰 권한 확인
3. 노션 페이지 ID가 올바른지 확인

### 분석 실패

**문제**: `❌ [클라이언트명] 분석 실패`

**해결**:
1. CSV 파일 형식 확인 (캠페인 레벨 vs 광고 레벨)
2. `analyze_meta_ad_full.py` 스크립트 로그 확인

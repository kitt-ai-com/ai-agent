# 전환 지표 및 ROAS 분석 가이드

## 📊 추가된 성과 지표

### 새로 추가된 지표

| 지표 | 설명 | 계산 방법 | 목표 |
|------|------|----------|------|
| **전환수** | 광고로 인한 전환 건수 | Meta Ads "결과" 컬럼 | 많을수록 좋음 |
| **CVR** | 전환율 | (전환수 / 클릭수) × 100 | 1% 이상 |
| **CPA** | 전환당 비용 | 지출 / 전환수 | KRW: ₩50,000 미만<br>USD: $50 미만 |

### 기존 지표

| 지표 | 설명 | 목표 |
|------|------|------|
| **CTR** | 클릭률 | 1% 이상 |
| **CPC** | 클릭당 비용 | KRW: ₩2,000 미만<br>USD: $2 미만 |
| **CPM** | 1,000회 노출당 비용 | KRW: ₩20,000 미만<br>USD: $20 미만 |

## 🎯 분석 로직

### CVR 체크 조건

```python
# 클릭이 10회 이상일 때만 CVR 체크
if (clicks > 10) and (CVR < 1.0%):
    → ROAS OFF 대상
```

**이유**: 클릭 수가 적을 때는 통계적으로 의미 없음

### CPA 체크 조건

```python
# 전환이 있을 때만 CPA 체크
if (conversions > 0) and (CPA > threshold):
    → ROAS OFF 대상
```

**이유**: 전환이 0건이면 CPA를 계산할 수 없음

## 🔮 향후 확장: ROAS (투자 수익률)

### ROAS란?

**ROAS (Return on Ad Spend)** = 구매전환값 / 광고비

예시:
- 광고비: ₩100,000
- 구매전환값: ₩500,000
- ROAS: 5.0 (500%)

### 구현 준비사항

현재 Meta Ads CSV에 구매전환값 데이터가 필요합니다:

**필요한 컬럼**:
- `구매 전환값` 또는 `Purchase Conversion Value`
- `구매 수` 또는 `Purchases`

**CSV 다운로드 시 설정**:
1. Meta Ads Manager → 광고 리포트
2. 열 맞춤설정 → 전환 섹션
3. 다음 지표 추가:
   - 구매 (웹사이트)
   - 구매 전환값 (웹사이트)
   - ROAS (웹사이트 구매)

### 코드 준비 완료

```python
# 향후 추가 시 사용할 코드
if '구매 전환값' in df.columns:
    df['ROAS'] = df['구매 전환값'] / df[spend_col]

    # ROAS OFF 기준 추가
    roas_off = df[
        (df['ROAS'] < 2.0)  # ROAS 200% 미만
    ]
```

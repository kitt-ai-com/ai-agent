# 네놈마켓 B2B 플랫폼 제안서 프롬프트

## 프로젝트 정보
- **클라이언트**: 네놈마켓
- **프로젝트명**: AI 채팅 기반 B2B 도매/유통 관리 플랫폼
- **개발 기간**: 6주
- **개발 비용**: 700만원
- **월 유지보수**: 20만원

## 제안서 구성 요소

### 1. 표지
- 제안서 제목
- 클라이언트명
- 제안 일자
- 제안사 정보

### 2. 프로젝트 개요
- 서비스 소개
- 핵심 가치 제안
- 타겟 사용자 (플랫폼 관리자, 셀러)

### 3. 제안 배경 및 목적
- 현재 B2B 유통 시장의 문제점
  - 엑셀 기반 수작업
  - 실시간 가격 변동 대응 어려움
  - 다채널 주문 통합 관리 복잡
  - 공급사 커뮤니케이션 비효율
- 네놈마켓의 솔루션
  - AI 채팅으로 자연어 업무 처리
  - 실시간 가격 관리
  - 자동 주문 처리
  - 스마트 발주 시스템

### 4. 핵심 기능
#### 4.1 플랫폼 관리자
- 상품 관리 (등록, 가격 설정, 재고 관리)
- 주문 관리 (접수, 취합, 발주서 생성)
- 배송 관리 (송장 매칭, 택배 API 연동)
- 회원 관리 (셀러 승인, 등급 관리)
- 정산/매출 관리 (수수료 계산, 통계)
- C/S 관리 (문의, 클레임)

#### 4.2 셀러
- 상품 조회 (등급별 가격)
- 주문 관리 (엑셀 업로드, 현황 조회)
- 송장 확인
- 클레임 요청
- 내 통계 조회

#### 4.3 AI 채팅 인터페이스
- 자연어 명령 처리
- 파일 업로드/다운로드
- 자동 알림
- 역할별 권한 제어

### 5. 기술 스택
- **프론트엔드**: Next.js 15, TypeScript, Tailwind CSS, shadcn/ui
- **백엔드**: Next.js API Routes, PostgreSQL, Prisma
- **AI**: Claude API (자연어 처리)
- **인증**: NextAuth.js
- **실시간**: WebSocket
- **배포**: Vercel (웹), Railway/Render (WebSocket), Supabase (DB/Storage)
- **외부 연동**: CJ대한통운 API, 카카오 알림톡 API

### 6. 개발 Phase (6주)
#### Phase 1: 기반 구축 (1주)
- DB 스키마 설계
- 인증 시스템
- 등급/수수료 시스템

#### Phase 2: 핵심 API (1.5주)
- 상품 CRUD
- 주문 CRUD
- 송장 매칭
- 발주서 생성

#### Phase 3: 웹 대시보드 (1.5주)
- 관리자 대시보드
- 셀러 대시보드
- 반응형 UI

#### Phase 4: AI 채팅 (1주)
- 채팅 UI
- Claude API 연동
- 파일 처리

#### Phase 5: 고급 기능 (0.5주)
- 택배 API 연동
- AI 자동 매핑
- 알림톡

#### Phase 6: 테스트 & 배포 (0.5주)
- 통합 테스트
- 보안 점검
- 운영 배포

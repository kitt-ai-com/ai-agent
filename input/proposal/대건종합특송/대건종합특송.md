# 대건종합특송 스마트 배차 관리 시스템 PRD

## 📌 프로젝트 개요

**고객사**: 대건종합특송
**사업 분야**: 대형차 택배 배차 관리
**작성일**: 2026년 3월 3일
**프로젝트 유형**: 물류 배차 관리 플랫폼 구축

---

## 🎯 비즈니스 목표

### 현재 문제점

1. **분산된 업무 환경**
   - 카카오톡으로 견적 요청 접수
   - 전국24시콜화물, 화물맨, 원콜, 인성데이타 등 4개 플랫폼을 개별적으로 확인
   - 각 플랫폼에 동일한 정보를 반복 입력
   - 수동으로 견적 비교

2. **메뉴얼 접근성 문제**
   - 부서별 업무 메뉴얼이 문서 형태로 산재
   - 필요한 정보를 찾는데 시간 소모
   - 신입 직원 교육 부담

3. **업무 비효율**
   - 플랫폼 조회 시간: 20분/건
   - 메뉴얼 확인 시간: 10분/회
   - 일평균 처리 건수: 10건

### 목표

1. **업무 시간 75% 단축**: 플랫폼 조회 시간 20분 → 5분
2. **처리량 150% 증가**: 일평균 10건 → 25건
3. **메뉴얼 접근성 90% 향상**: AI 챗봇을 통한 즉각 응답
4. **데이터 자동화**: 파일 업로드 및 분류 자동화

## 📋 요구사항 (이미지 분석 기반)

### 1. 업무 흐름 체계 (IMG_2813, IMG_2814)

#### 1.1 견적 요청 접수
- **입력**: 관계사, 프랜차이즈 업체가 카카오톡으로 배차 견적서 요청
- **파일 형태**: 이미지 파일 (JPG, PNG, PDF)
- **요구사항**:
  - 카톡 파일을 시스템에 업로드
  - 자동 분류 및 저장 (날짜별, 고객사별)
  - 처리 상태 관리 (접수/처리중/완료)

#### 1.2 메뉴얼 참조
- **현황**: 매 요일마다 부서별 업무 현황 설명 이미지 참고
- **요구사항**:
  - AI 챗봇 연동으로 실시간 메뉴얼 제공
  - 부서별, 요일별 메뉴얼 데이터베이스 구축
  - 질의응답 형태로 빠른 정보 확인

#### 1.3 4개 플랫폼 견적 조회
- **플랫폼 목록**:
  1. 전국24시콜화물 (www.callfreight.co.kr)
  2. 화물맨 (www.hwamulman.co.kr)
  3. 원콜 (www.1callcargo.co.kr)
  4. 인성데이타 (www.insung.co.kr)

- **현재 프로세스**: 각 사이트에 개별 접속하여 견적 조회
- **요구사항**:
  - 한 번의 입력으로 4개 플랫폼 동시 조회
  - 실시간 견적 비교 대시보드
  - 가격 오름차순 정렬

#### 1.4 견적서 제공
- **출력**: 고객사에 최종 견적서 제공
- **요구사항**: 견적 내역 저장 및 이력 관리

#### 1.5 추가 업무 흐름 (IMG_2814)
- 계약서 작성 전 고객사와 예약 견적 조건 부합 여부 확인
- 계약서 회사명/회사코드 지정 시 정확한 회사명/주소 검증
- 내부 기본값 가격 대비 견적가 산정 기준 관리
- 적재 시점 드라이버 미정차 시 집하지연 공지
- 무게 초과 시 초과분 사전고지 후 처리

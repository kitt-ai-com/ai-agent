# 메타광고 컨텐츠 제작 빠른 시작 가이드

## 🎯 목표

메타광고에서 **판매 극대화 + ROAS 3.0 이상 달성**을 위한 광고 소재를 체계적으로 제작합니다.

---

## 📋 준비물

1. **제품 정보**: 제품명, 가격, 특징, 프로모션
2. **리뷰 데이터**: 스마트스토어/쿠팡 리뷰 최소 50개
3. **경쟁사 광고**: 메타광고 라이브러리에서 수집한 광고 10개 이상

---

## 🚀 5단계 프로세스

### Step 1: 폴더 및 파일 확인 ✅

이미 생성된 폴더 구조:

```
ai-agent/
├── meta-contents.md                   # 📘 메인 지침서 (AI 프롬프트 포함)
├── QUICKSTART-META-AD.md             # 📄 이 파일 (빠른 시작 가이드)
├── input/meta-ad/                    # 📥 작업 진행 파일
│   ├── product-info-template.md
│   ├── smartstore-reviews-template.md
│   └── competitor-ads-template.md
└── output/meta-ad/                   # 📤 결과물 파일
    └── README.md
```

---

### Step 2: Input 파일 작성 (30-60분)

#### 2-1. 제품 정보 작성

```bash
cd input/meta-ad
cp product-info-template.md product-info-[제품명].md
```

[product-info-[제품명].md](input/meta-ad/product-info-template.md) 파일을 열어서 [ ] 부분을 실제 정보로 채우세요.

**필수 입력 항목**:
- 제품명, 카테고리, 가격
- 주요 특징 5가지
- 타겟 고객 (연령, 성별, 라이프스타일)
- 경쟁 우위 3가지

---

#### 2-2. 리뷰 수집 및 분석

```bash
cp smartstore-reviews-template.md smartstore-reviews-[제품명].md
```

**리뷰 수집 방법**:
1. 스마트스토어 또는 쿠팡에서 리뷰 최소 50개 수집
2. 긍정 리뷰 Best 20개 복사
3. 부정 리뷰 주요 불만 5개 복사
4. 키워드 빈도 분석 (품질, 가성비, 배송 등)

**핵심**:
- 광고에 인용할 수 있는 리뷰 선별
- 사회적 증거 (평점, 판매량, 재구매율) 정리

---

#### 2-3. 경쟁사 광고 분석

```bash
cp competitor-ads-template.md competitor-ads-[제품명].md
```

**광고 수집 방법**:
1. [메타광고 라이브러리](https://www.facebook.com/ads/library/) 접속
2. "모든 광고" 탭에서 제품 카테고리 검색
3. 최소 10개 광고 수집 (문구, 이미지, CTA 포함)
4. 오래 게재 중인 광고 = 성과 좋은 광고 → 우선 분석

**분석 항목**:
- 문구 스타일 (짧은 문구 vs 롱카피)
- 주요 키워드 (할인, 한정, 무료배송 등)
- 감정 자극 전략 (긴급성, 혜택, 사회적 증거)

---

### Step 3: AI 프롬프트 실행 (30분)

[meta-contents.md](meta-contents.md) 파일을 열어서 프롬프트를 순서대로 실행하세요.

#### 프롬프트 1: 소구점 및 리뷰 분석

**Input**: `product-info-[제품명].md` + `smartstore-reviews-[제품명].md`

**Output**: `output/meta-ad/analysis-report.md`

**결과물**:
- 핵심 소구점 3가지
- 리뷰 기반 긍정 요소 5가지
- 타겟 고객 정의
- 광고 강조 포인트

---

#### 프롬프트 2: 경쟁사 광고 벤치마킹

**Input**: `competitor-ads-[제품명].md`

**Output**: `analysis-report.md` (추가)

**결과물**:
- 성공 광고 패턴 분석
- 주요 키워드 및 CTA
- 우리 제품 적용 전략 3가지

---

#### 프롬프트 3: 버전1 광고 소재 (짧은 문구)

**Input**: `analysis-report.md`

**Output**: `output/meta-ad/ad-copy-v1.md`

**결과물**:
- 30자 이상 짧은 문구 × 10개
- 제목 × 10개
- 다양한 스타일 (질문형, 혜택 강조형, 긴급성 강조형 등)

---

#### 프롬프트 4: 버전2 광고 소재 (롱카피)

**Input**: `analysis-report.md`

**Output**: `output/meta-ad/ad-copy-v2.md`

**결과물**:
- 100자 이상 롱카피 × 10개
- 제목 × 10개
- 스토리텔링 구조 (문제 → 공감 → 솔루션 → 혜택 → CTA)

---

#### 프롬프트 5: ROAS 최적화 전략

**Input**: `ad-copy-v1.md` + `ad-copy-v2.md`

**Output**: `output/meta-ad/roas-strategy.md`

**결과물**:
- A/B 테스트 전략
- 타겟팅 설정 추천
- 광고 게재 위치 최적화
- 입찰 전략
- 단계별 실행 계획 (1주차, 2주차, 3주차)

---

### Step 4: 최종 정리 및 실행 (20분)

모든 결과물을 바탕으로 `output/meta-ad/final-recommendation.md` 작성:

```markdown
# 최종 광고 소재 추천

## 1차 테스트 소재 (3개)
- 버전1 소재 #1: [문구] → 타겟: [25-34세 여성]
- 버전1 소재 #3: [문구] → 타겟: [35-44세 남성]
- 버전2 소재 #2: [문구] → 타겟: [관심사 타겟팅]

## A/B 테스트 계획
- 기간: 3일
- 예산: 일 5만원
- 측정 지표: CTR, CPC, 전환율

## 타겟팅 설정
- 1차 타겟: [연령대, 성별, 관심사]
- 2차 타겟: [Lookalike 1%]

## 성과 목표
- CTR: 2% 이상
- ROAS: 3.0 이상
```

---

### Step 5: 메타광고 캠페인 세팅 (30분)

1. **Meta Business Suite** 또는 **광고 관리자** 접속
2. 캠페인 생성: "판매" 목표 선택
3. 광고 세트 생성: 타겟팅 설정 (연령, 성별, 관심사)
4. 광고 소재 업로드:
   - 이미지/영상 (권장 크기: 1080x1080px)
   - 문구 및 제목 (생성한 소재 사용)
   - CTA 버튼 (지금 쇼핑하기, 자세히 보기 등)
5. 예산 및 일정 설정
6. 광고 게시 및 성과 모니터링

---

## 📊 성과 측정 및 최적화

### 1주차: 데이터 수집
- CTR, CPC, 전환율 측정
- 성과 좋은 소재 식별

### 2주차: 최적화
- 성과 낮은 광고 중단
- 성과 좋은 광고 예산 증액
- 타겟팅 조정 (관심사 추가/제외)

### 3주차: 확장
- Lookalike 타겟팅 추가
- 새로운 크리에이티브 테스트
- ROAS 3.0 이상 유지

## 🆘 문제 해결 (FAQ)

### Q1. 리뷰가 부족한 경우 (50개 미만)
→ 경쟁사 제품 리뷰도 참고하여 카테고리 공통 니즈 파악

### Q2. 경쟁사 광고를 찾을 수 없는 경우
→ 상위 카테고리로 검색 범위 확대 (예: "가죽 지갑" → "지갑")

### Q3. AI 생성 문구가 부자연스러운 경우
→ 프롬프트에 "자연스러운 구어체로 작성" 추가 요청

### Q4. ROAS가 목표에 미달하는 경우
→ 타겟팅 좁히기 (관심사 세분화), 크리에이티브 교체, 입찰 전략 변경

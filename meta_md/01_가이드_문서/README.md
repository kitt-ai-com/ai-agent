### 2. [메타광고_빠른시작.md](메타광고_빠른시작.md) - 5분 빠른 시작 ⚡

**용도**: 즉시 실행 가이드
**대상**: 바쁜 사용자, 초보자

**결과물**: 분석 리포트 3종 + 즉시 실행 액션

### 3. [메타광고_통합_분석_및_제작_가이드.md](메타광고_통합_분석_및_제작_가이드.md) - 전체 프로세스 📗

**용도**: 분석부터 신규 광고 제작까지 완전 가이드
**대상**: 고급 사용자, 마케터
**소요 시간**: 1-2시간

**3단계 프로세스**:
- **STEP 1**: 기존 광고 성과 분석 (30분)
- **STEP 2**: 개선된 신규 광고 제작 (40분)
- **STEP 3**: 실행 및 모니터링 (30분)

**결과물**:
- 분석 리포트 3종
- 광고 소재 20개 (짧은 문구 10 + 롱카피 10)
- ROAS 최적화 전략
- 최종 추천 광고 3-5개

**추천**: 신규 광고 제작까지 필요한 경우!

### 4. [QUICKSTART-META-AD.md](QUICKSTART-META-AD.md) - 광고 제작 가이드 🎨

**용도**: 신규 광고 소재 제작 전문 가이드
**대상**: 마케터, 광고 제작자
**소요 시간**: 2-3시간

**5단계 프로세스**:
1. 제품 정보 입력
2. 고객 리뷰 수집 및 분석
3. 경쟁사 광고 벤치마킹
4. AI 프롬프트 실행 (자동 소재 생성)
5. 메타광고 캠페인 세팅

**결과물**:
- 광고 소재 20개 (다양한 스타일)
- A/B 테스트 계획
- 타겟팅 전략
- 예산 배분 계획

**추천**: 기존 광고 없이 새로 시작하는 경우!

## 📋 준비물 (단 1개만!)

- [ ] **메타광고 CSV 파일** (최근 30일 데이터)

### CSV 다운로드 방법 (3분)

1. [메타 광고 관리자](https://business.facebook.com/adsmanager) 접속
2. 우측 상단 `...` → **내보내기** 클릭
3. 날짜 범위: **최근 30일** 선택
4. 포함 항목:
   - ✅ 광고 이름
   - ✅ 노출, 클릭, 지출금액
   - ✅ 전환 데이터 (리드, 구매 등)
   - ✅ CTR, CPC, CPM
5. 형식: **CSV** 선택 → **내보내기**
6. 다운로드 완료 후 파일 위치 확인

## 💡 바로 실행할 액션 (리포트 확인 후)

### 오늘 즉시 (5분)

리포트에서 다음 3가지 확인:

1. **중단할 광고** (저성과)
   - "개선 필요 광고 TOP 3" 섹션 확인
   - 메타 광고 관리자에서 해당 광고 중단

2. **증액할 광고** (고성과)
   - "최고 성과 광고 TOP 3" 섹션 확인
   - 예산 +50% 증액

3. **신규 제작 방향**
   - 고성과 광고의 포맷/메시지 확인
   - 유사한 광고 3-5개 제작 계획

## 🆘 문제 해결

### Q1. "pandas 모듈을 찾을 수 없습니다" 오류

```bash
# Python 라이브러리 설치
pip3 install pandas numpy
```

### Q2. "CSV 파일을 읽을 수 없습니다" 오류

**원인**: CSV 인코딩 문제

**해결**:
1. CSV 파일을 Excel로 열기
2. **다른 이름으로 저장** 클릭
3. 형식: **CSV UTF-8 (쉼표로 분리)** 선택
4. 저장 후 다시 실행

### Q3. "파일을 찾을 수 없습니다" 오류

**해결**:
```bash
# 현재 디렉토리 확인
pwd

# ai-agent 폴더로 이동
cd /Users/jasonmac/kitt_agent/ai-agent

# 다시 실행
python3 scripts/analyze_meta_ad_full.py
```

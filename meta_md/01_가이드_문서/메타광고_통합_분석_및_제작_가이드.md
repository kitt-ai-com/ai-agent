# STEP 1: 기존 광고 성과 분석 (30분)

## 1-1. 광고 데이터 준비

### CSV 파일 내보내기 방법

1. **메타 광고 관리자** 접속
2. 우측 상단 `...` 클릭 → `내보내기` 선택
3. 날짜 범위: 최근 **30일** (최소 7일)
4. 포함 항목 선택:
   - ✅ 캠페인/광고세트/광고 구조
   - ✅ 노출, 클릭, 지출금액
   - ✅ 전환 데이터 (구매, 가입, 리드 등)
   - ✅ CTR, CPC, CPM 등 주요 지표
5. 형식: `.csv` 선택 → `내보내기`

## 1-4. 핵심 인사이트 확인

분석 리포트에서 다음 항목을 확인하세요:

### ✅ 확인할 핵심 지표

| 항목 | 확인 내용 |
|------|----------|
| **고성과 광고** | 리드당 비용 또는 ROAS가 가장 좋은 광고 Top 3 |
| **저성과 광고** | 즉시 중단해야 할 광고 (비효율적) |
| **소재 타입** | 이미지 vs 영상 vs 캐러셀 중 어떤 게 잘 됐는지 |
| **메시지 패턴** | 어떤 카피/메시지가 효과적이었는지 |
| **타겟팅** | 연령대/성별/관심사별 성과 차이 |

### 📊 예시 인사이트

```
✅ 인사이트 1: 영상 광고가 이미지 대비 리드당 비용 40% 낮음
   → 액션: 영상 소재 5개 추가 제작

✅ 인사이트 2: "무료 배송" 메시지가 CTR 2배 높음
   → 액션: 모든 신규 광고에 무료 배송 강조

❌ 문제점 1: "기술 용어" 사용 광고의 전환율 낮음
   → 액션: 고객 언어로 메시지 재작성
```

## 3-2. A/B 테스트 계획

### 테스트 구조

```
광고 세트 A: 짧은 문구 광고 3개
  - 광고 A1: 질문형 메시지
  - 광고 A2: 혜택 강조형
  - 광고 A3: 긴급성 강조형

광고 세트 B: 롱카피 광고 2개
  - 광고 B1: 스토리텔링형
  - 광고 B2: 사회적 증거형

각 광고 세트에 동일 예산 배정 (일 25,000원씩)
```

### 측정 지표

| 지표 | 목표 | 판단 기준 |
|------|------|----------|
| **CTR** | 2% 이상 | 🟢 우수: 2.5%+, 🟡 보통: 1.5-2.5%, 🔴 나쁨: 1.5% 미만 |
| **CPC** | 500-1,000원 | 낮을수록 좋음 |
| **전환율** | 1-3% | 클릭 대비 전환 |
| **ROAS** | 3.0 이상 | 매출 ÷ 광고비 |
| **리드당 비용** | [목표 설정] | 분석 리포트 평균 대비 낮아야 함 |

→ 제품 정보만으로도 실행 가능 (리뷰/경쟁사는 선택 사항)

```bash
python3 generate_meta_ads.py \
  --analysis "output/meta-ad/분석_리포트.md" \
  --product "input/meta-ad/product-info-[업체명].md"
```

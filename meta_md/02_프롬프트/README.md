## 📁 프롬프트 목록 (2개)

### 1. [meta-ad-analysis-prompt.md](meta-ad-analysis-prompt.md) - 성과 분석 프롬프트

**용도**: 기존 메타광고 성과 자동 분석
**입력**: 메타광고 CSV 데이터
**출력**: 분석 리포트 + 개선 전략

**주요 기능**:
- 전체 성과 요약
- 캠페인별 성과 비교
- 고성과/저성과 광고 식별
- 즉시 실행 액션 제안
- 예상 개선 효과 시뮬레이션

**사용 시점**:
- 기존 광고 성과를 분석하고 싶을 때
- 어떤 광고가 잘 되는지 파악하고 싶을 때
- 예산 최적화가 필요할 때

**실행 방법**:
```bash
# 자동 실행 (권장)
python3 ../03_스크립트/analyze_meta_ad.py --csv "광고데이터.csv"

# 수동 실행
# 1. meta-ad-analysis-prompt.md 파일 열기
# 2. 프롬프트 복사
# 3. Claude 또는 ChatGPT에 붙여넣기
# 4. CSV 데이터 함께 제공
```

---

### 2. [meta-contents.md](meta-contents.md) - 광고 소재 제작 프롬프트

**용도**: 신규 메타광고 소재 자동 생성
**입력**: 제품 정보 + 리뷰 데이터 + 경쟁사 광고
**출력**: 광고 소재 20개 + 전략

**주요 기능**:
- 짧은 문구 광고 10개 생성
- 롱카피 광고 10개 생성
- 다양한 스타일 (질문형, 혜택형, 긴급성 등)
- A/B 테스트 계획
- 타겟팅 전략 수립
- ROAS 최적화 전략

**사용 시점**:
- 신규 광고 소재가 필요할 때
- 기존 광고 메시지를 개선하고 싶을 때
- A/B 테스트용 다양한 버전이 필요할 때

**실행 방법**:
```bash
# 준비물:
# 1. 제품 정보 (input/meta-ad/product-info-[제품명].md)
# 2. 리뷰 데이터 (input/meta-ad/smartstore-reviews-[제품명].md)
# 3. 경쟁사 광고 (input/meta-ad/competitor-ads-[제품명].md)

# 실행:
# 1. meta-contents.md 파일 열기
# 2. 5단계 프롬프트 순서대로 실행
# 3. 각 단계 결과를 다음 프롬프트에 활용
```

## 1. 데이터 수집 단계

### 1.1 데이터 입력 방식 확인

사용자에게 다음 중 하나를 선택하도록 안내:

| 방식 | 장점 | 필요 정보 |
|------|------|----------|
| **CSV 파일** | 정확하고 상세한 분석 가능 | 메타 광고 관리자에서 Export한 CSV |
| **스크린샷** | 빠르고 간편 | 광고 관리자 대시보드 캡처 |
| **수동 입력** | 민감 정보 보호 | 주요 지표만 텍스트로 입력 |

### 1.2 CSV 파일 처리 (방식 1 선택 시)

```python
# CSV 파일을 읽고 주요 컬럼 확인
import pandas as pd

df = pd.read_csv('광고_데이터.csv')
required_columns = [
    '캠페인명', '광고세트명', '광고명',
    '노출', '클릭', '지출금액', '구매', '매출',
    'CTR', 'CPC', 'CPM', 'ROAS'
]

# 데이터 요약
print(f"분석 기간: {df['날짜'].min()} ~ {df['날짜'].max()}")
print(f"총 캠페인 수: {df['캠페인명'].nunique()}")
print(f"총 지출: {df['지출금액'].sum():,.0f}원")
```

### 1.3 스크린샷 분석 (방식 2 선택 시)

이미지를 읽고 주요 지표를 추출:
- OCR 또는 이미지 인식으로 숫자 추출
- 주요 지표: 노출, 클릭, 지출, 전환, ROAS 등

### 1.4 수동 입력 템플릿 (방식 3 선택 시)

사용자에게 다음 정보를 요청:


## 2. 분석 프레임워크

### 2.1 필수 분석 항목

#### A. 전체 성과 요약

| 지표 | 현재 값 | 업계 평균 | 평가 |
|------|---------|----------|------|
| **ROAS** | X% | 200-400% | 🔴/🟡/🟢 |
| **CTR** | X% | 1-2% | 🔴/🟡/🟢 |
| **CPC** | X원 | 500-1,500원 | 🔴/🟡/🟢 |
| **전환율** | X% | 1-3% | 🔴/🟡/🟢 |
| **CPA** | X원 | - | 🔴/🟡/🟢 |

**평가 기준:**
- 🟢 우수 (상위 25%)
- 🟡 보통 (평균 수준)
- 🔴 개선 필요 (하위 25%)

#### B. 캠페인별 성과 비교

각 캠페인의 주요 지표를 비교하여:
1. **Best Performer** - 가장 성과 좋은 캠페인
2. **Worst Performer** - 개선이 필요한 캠페인
3. **Budget Allocation** - 예산 재분배 제안

#### C. 시계열 트렌드 분석 (데이터가 충분한 경우)

- 일별/주별 성과 변화
- 요일별 패턴 (주중 vs 주말)
- 특이점 감지 (급등/급락)

#### D. 타겟 오디언스 분석

- 연령대별 성과
- 성별 성과
- 지역별 성과
- 디바이스별 성과 (모바일 vs 데스크톱)

#### E. 크리에이티브 분석

- 소재 유형별 성과 (이미지/동영상/캐러셀)
- 카피 메시지별 성과
- CTA 버튼별 성과

## 3. 인사이트 도출

### 3.1 긍정적 인사이트 (강점)

**템플릿:**
```
✅ 강점 1: [구체적 지표와 수치]
   → 이유: [왜 좋은 성과가 나왔는지]
   → 활용: [이 강점을 어떻게 확대할지]

✅ 강점 2: ...
```

**예시:**
```
✅ 강점 1: 20-30대 여성 타겟의 ROAS가 450%로 전체 평균(280%)보다 61% 높음
   → 이유: 인스타그램 스토리 광고 + 인플루언서 협업 이미지 소재 효과
   → 활용: 이 타겟에 예산 30% 추가 배분, 유사 타겟 확장
```

### 3.2 개선 필요 영역 (약점)

**템플릿:**
```
🔴 문제점 1: [구체적 지표와 수치]
   → 원인: [왜 성과가 낮은지 가설]
   → 해결: [구체적 개선 방안]
   → 우선순위: 높음/중간/낮음

🔴 문제점 2: ...
```

**예시:**
```
🔴 문제점 1: 데스크톱 광고의 CTR이 0.3%로 모바일(1.8%)의 1/6 수준
   → 원인: 이미지 소재가 모바일에 최적화되어 있고, 데스크톱 랜딩페이지 로딩 속도 느림
   → 해결:
      1. 데스크톱용 와이드 이미지 소재 제작
      2. 랜딩페이지 로딩 속도 개선 (3초 → 1초 목표)
      3. 또는 데스크톱 타겟팅 중단 고려
   → 우선순위: 높음 (전체 지출의 25% 차지)
```

### 3.3 숨겨진 기회 발견

**찾아야 할 것:**
- 저평가된 고성과 오디언스
- 특정 시간대/요일의 높은 전환율
- 예상 밖의 관심사 타겟 성과
- 경쟁이 낮은 키워드/타겟

### 4.1 즉시 실행 (1-3일 내)

**우선순위 높음, 빠른 효과**

| 액션 | 목표 | 예상 효과 | 담당 |
|------|------|----------|------|
| 저성과 광고 3개 중단 | 예산 낭비 방지 | 월 XX만원 절감 | 마케터 |

| 랜딩페이지 A/B 테스트 시작 | 전환율 개선 | 전환율 +0.5%p | 개발팀 |

### 4.2 단기 실행 (1-2주 내)

**중요하지만 준비 필요**

- 새 타겟 오디언스 3개 테스트 (예산: 각 10만원)

- 리타게팅 캠페인 세팅 (웹사이트 방문자 대상)
- 경쟁사 분석 및 벤치마크

### 4.3 중기 실행 (1개월 내)

**구조적 개선**

- 퍼널 전체 최적화 (인지 → 고려 → 전환)
- 크리에이티브 A/B 테스트 프로세스 구축
- 자동화 규칙 설정 (예: ROAS 150% 미만 시 자동 중단)
- 월간 리포트 자동화 대시보드 구축

## 7. 템플릿 사용 예시

### 입력 예시

```
## 광고 캠페인 기본 정보
- 캠페인명: 신상품 런칭 프로모션
- 분석 기간: 2026.02.01 ~ 2026.02.28
- 광고 목적: 전환 (구매)

## 주요 지표
- 총 지출: 5,000,000원
- 노출수: 2,500,000
- 클릭수: 30,000
- CTR: 1.2%
- CPC: 167원
- CPM: 2,000원

## 전환 데이터
- 구매 전환수: 450건
- 전환율: 1.5%
- CPA: 11,111원
- 총 매출: 13,500,000원
- ROAS: 270%

## 타겟팅 정보
- 타겟 연령대: 25-40세
- 타겟 성별: 여성 70%, 남성 30%
- 관심사: 건강, 웰니스, 유기농
- 지역: 수도권
```

### 출력 예시

```markdown
# 메타광고 성과 분석 리포트

**분석 기간:** 2026.02.01 ~ 2026.02.28 (28일)
**총 지출:** 5,000,000원
**총 매출:** 13,500,000원
**ROAS:** 270% 🟡

## 전체 성과 평가

| 지표 | 실적 | 업계 평균 | 평가 |
|------|------|----------|------|
| ROAS | 270% | 250-400% | 🟡 보통 |
| CTR | 1.2% | 1.0-1.5% | 🟢 우수 |
| CPC | 167원 | 500-1,000원 | 🟢 우수 (저비용) |
| 전환율 | 1.5% | 1-3% | 🟢 우수 |
| CPA | 11,111원 | - | 🟡 (제품 가격 대비 평가 필요) |

**종합 평가:**
트래픽 확보(CTR, CPC)는 매우 우수하나, ROAS가 평균 수준에 머물러 있음.
전환율은 양호하므로, 객단가 향상 또는 재구매 유도가 핵심.

## 핵심 인사이트 3가지

1. 💡 **CPC가 업계 평균의 1/3 수준으로 매우 효율적**
   - 타겟팅이 정확하고 소재 퀄리티가 높음
   - 경쟁이 낮은 틈새 타겟 공략 성공

2. 💡 **전환율 1.5%는 우수하지만, ROAS는 보통 수준**
   - 객단가가 30,000원으로 낮은 편 (13,500,000원 ÷ 450건)
   - 추가 구매 유도 또는 객단가 상승 전략 필요

3. 💡 **여성 타겟 비중 70%로 편중**
   - 남성 타겟 30%의 성과가 여성 대비 어떤지 분석 필요
   - 잠재적으로 남성 시장 확대 가능성

## 즉시 실행 액션 3가지

1. ⚡ **객단가 상승 전략 (크로스셀/업셀)**
   - 연관 상품 추천, 세트 상품 구성
   - 예상 효과: 객단가 30,000원 → 40,000원 (↑33%)
   - → ROAS 270% → 360% (↑90%p)

2. ⚡ **고성과 타겟에 예산 집중**
   - 현재 가장 성과 좋은 오디언스 세그먼트 파악 후 예산 +50%
   - 예상 효과: 전체 ROAS ↑10-15%p

3. ⚡ **리타게팅 캠페인 즉시 시작**
   - 웹사이트 방문자 중 미구매자 대상 (예산: 월 100만원)
   - 예상 효과: CPA 50% 감소, ROAS +100%p

## 단기 개선 제안 (1-2주)

1. **남성 타겟 별도 캠페인 테스트**
   - 남성 전용 소재/카피 제작
   - 예산: 50만원 (테스트)

2. **동영상 광고 추가**
   - 현재 이미지만 사용 중이라면 동영상 제작
   - 평균 CTR 30% 향상 기대

3. **구매 고객 Lookalike 오디언스 생성**
   - 기존 450명 구매 고객 기반
```

### 주요 용어 정의

- **ROAS (Return on Ad Spend)**: 광고 수익률 = (매출 ÷ 광고비) × 100
- **CTR (Click-Through Rate)**: 클릭률 = (클릭수 ÷ 노출수) × 100
- **CPC (Cost Per Click)**: 클릭당 비용 = 지출금액 ÷ 클릭수
- **CPM (Cost Per Mille)**: 1000회 노출당 비용 = (지출금액 ÷ 노출수) × 1000
- **CPA (Cost Per Action)**: 전환당 비용 = 지출금액 ÷ 전환수
- **전환율 (Conversion Rate)**: (전환수 ÷ 클릭수) × 100

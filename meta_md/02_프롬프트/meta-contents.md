# 메타광고 컨텐츠 제작 지침서

## 📁 폴더 구조

```
ai-agent/
├── input/meta-ad/           # 작업 진행 파일 (제품 정보, 리뷰 데이터 등)
└── output/meta-ad/          # 결과물 파일 (최종 광고 소재)
```

---

## 🎯 메타광고 컨텐츠 작업 프로세스

### 1단계: 제품 분석 (Input)
- 제품명, 카테고리, 가격, 특징
- 스마트스토어 리뷰 데이터 수집
- 경쟁사 광고 분석 (메타광고 라이브러리)

### 2단계: 컨텐츠 제작 (Output)
- 버전1: 짧은 문구 (30자 이상) + 제목 × 10개
- 버전2: 롱카피 (100자 이상) + 제목 × 10개
- 소구점 정리
- 리뷰 기반 강조 포인트

### 3단계: 최적화 전략
- 판매 전환 극대화
- 조회수 증대
- ROAS (광고비 대비 매출) 최적화

```
당신은 메타광고 전문 마케터입니다. 다음 제품 정보와 스마트스토어 리뷰를 분석하여 광고 전략을 수립해주세요.

**제품 정보:**
- 제품명: [제품명]
- 카테고리: [카테고리]
- 가격: [가격]
- 주요 특징: [특징 1, 특징 2, 특징 3]

**스마트스토어 리뷰 데이터:**
[리뷰 내용 또는 리뷰 요약]

**분석 요청사항:**
1. 제품의 핵심 소구점 3가지 추출
2. 리뷰에서 가장 많이 언급된 긍정적 요소 5가지
3. 강조해야 할 제품 강점 (리뷰 기반)
4. 타겟 고객층 정의
5. 경쟁 우위 요소

**출력 형식:**

- 핵심 소구점 1: [내용]

## 리뷰 분석 (긍정 요소)
1. [요소 1] - 언급 빈도: X회
2. [요소 2] - 언급 빈도: X회
...

## 강조 포인트
- 광고에서 반드시 강조할 부분: [내용]
- 차별화 포인트: [내용]

## 타겟 고객
- 주요 타겟: [연령대, 성별, 관심사]
- 페인 포인트: [고객의 문제점]
```

### 프롬프트 5: ROAS 최적화 전략 수립

```
당신은 퍼포먼스 마케팅 전문가입니다. 다음 광고 소재를 바탕으로 ROAS(광고비 대비 매출) 극대화 전략을 수립해주세요.

**현재 상황:**
- 제품: [제품명]
- 목표: 판매 전환 + 조회수 증대 + ROAS 3.0 이상
- 예산: [월 예산]
- 타겟: [타겟 고객]

**전략 수립 요청:**
1. A/B 테스트 전략 (어떤 소재를 먼저 테스트할지)
2. 타겟팅 설정 추천 (맞춤 타겟, 관심사, lookalike)
3. 광고 게재 위치 최적화 (피드, 스토리, 릴스 등)
4. 입찰 전략 (전환 최적화 vs 도달 최적화)
5. 성과 측정 지표 (핵심 KPI)
6. 단계별 실행 계획 (1주차, 2주차, 3주차)

**출력 형식:**

## A/B 테스트 전략
- 테스트 1: [버전1 소재 1-3번] vs [버전1 소재 4-6번]
- 측정 지표: [CTR, CPC, 전환율]
- 기간: 3일

## 타겟팅 설정
- 1차 타겟: [연령대, 성별, 지역, 관심사]
- 2차 타겟 (lookalike): [기존 고객 기반 유사 타겟]
- 제외 타겟: [이미 구매한 고객 등]

## 광고 게재 위치
- 우선순위 1: [피드 / 스토리 / 릴스]
- 우선순위 2: [...]
- 제외: [Audience Network 등]

## 입찰 전략
- 초기 (1주차): [최저 단가 입찰 / 도달 최적화]
- 성과 확인 후 (2주차~): [전환 최적화]
- 일 예산: [금액]

## 핵심 KPI
- CTR 목표: X% 이상
- CPC 목표: X원 이하
- 전환율 목표: X% 이상
- ROAS 목표: 3.0 이상

## 단계별 실행 계획
### 1주차
- [실행 항목 1]
- [실행 항목 2]

### 2주차
- [성과 분석 및 최적화]

### 3주차
- [확장 전략]
```

## 🔄 작업 워크플로우

### Step 1: Input 파일 준비 (`input/meta-ad/`)
1. `product-info.md` - 제품 정보
2. `smartstore-reviews.csv` - 스마트스토어 리뷰 데이터
3. `competitor-ads.md` - 메타광고 라이브러리 분석 자료

### Step 2: AI 프롬프트 실행
1. 프롬프트 1 실행 → 소구점 분석 결과
2. 프롬프트 2 실행 → 경쟁사 광고 분석 결과
3. 프롬프트 3 실행 → 버전1 소재 10개
4. 프롬프트 4 실행 → 버전2 소재 10개
5. 프롬프트 5 실행 → ROAS 최적화 전략

### Step 3: Output 파일 생성 (`output/meta-ad/`)
1. `analysis-report.md` - 소구점 + 리뷰 + 경쟁사 분석 종합
2. `ad-copy-v1.md` - 버전1 소재 10개 (짧은 문구)
3. `ad-copy-v2.md` - 버전2 소재 10개 (롱카피)
4. `roas-strategy.md` - ROAS 극대화 전략서
5. `final-recommendation.md` - 최종 추천 소재 및 실행 계획

## 📊 성과 측정 및 최적화

### 주요 지표
- **CTR (클릭률)**: 2% 이상 목표
- **CPC (클릭당 비용)**: 업계 평균 이하
- **전환율**: 3% 이상 목표
- **ROAS**: 3.0 이상 (광고비 대비 3배 매출)
- **조회수**: 일 평균 X회 이상

### 최적화 액션
1. **CTR 낮음** → 제목/문구/이미지 변경
2. **전환율 낮음** → 랜딩 페이지 개선, CTA 변경
3. **ROAS 낮음** → 타겟팅 재설정, 입찰 전략 변경
4. **조회수 낮음** → 예산 증액, 타겟 확대

## 💡 추가 팁

### 고성과 광고 문구 패턴
1. **숫자 활용**: "30% 할인", "5만원 이상 무료배송"
2. **긴급성**: "오늘만", "한정 수량", "마감 임박"
3. **사회적 증거**: "누적 판매 1만개", "★4.9 만족도"
4. **질문형**: "아직도 XX 때문에 고민하세요?"
5. **혜택 강조**: "지금 구매 시 XX 증정"

### 이미지/영상 가이드
- 제품 중심: 깔끔한 배경, 제품 디테일 강조
- 라이프스타일: 제품 사용 장면, 고객 만족 표현
- 텍스트 오버레이: 핵심 메시지 3-5단어 이내
- 영상: 첫 3초가 핵심, 무음으로도 이해 가능해야 함

### 캠페인 구조 추천
```
캠페인: [제품명] 판매 캠페인
├── 광고세트 1: 관심사 타겟팅 (25-34세 여성)
│   ├── 광고 1-1: 버전1 소재 1 + 이미지 A
│   ├── 광고 1-2: 버전1 소재 2 + 이미지 B
│   └── 광고 1-3: 버전2 소재 1 + 영상 A
├── 광고세트 2: Lookalike 타겟팅 (구매자 유사)
│   ├── 광고 2-1: 버전1 소재 3 + 이미지 C
│   └── 광고 2-2: 버전2 소재 2 + 영상 B
└── 광고세트 3: 리타겟팅 (장바구니 이탈)
    ├── 광고 3-1: 긴급성 강조 문구 + 할인 쿠폰
    └── 광고 3-2: 리뷰 강조 문구 + 고객 후기
```

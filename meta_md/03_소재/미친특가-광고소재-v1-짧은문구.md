# 미친특가 광고 소재 - 버전1 (짧은 문구)

**제작일:** 2026-03-19
**컨셉:** 미친특가 프로모션
**타겟:** 클메 쇼핑몰 고객 (주부, 직장인, 일반 소비자)
**벤치마크:** 주부들_롱버전 (CTR 16.25%), 묵직한_영상 (CTR 15.10%)

## 소재 1: 긴급성 + 타겟 특정형 (주부)

**제목:** 주부님 보세요
**문구:** 마트 가지 마세요! 집에서 클릭 한 번이면 미친특가로 문앞까지 배송됩니다
**길이:** 39자
**스타일:** 타겟 특정 + 편의성 강조
**CTA:** 지금 바로 보기
**예상 CTR:** 14-16% (주부 타겟 효과)

**사용 이미지:**
- 주부가 스마트폰으로 쇼핑하는 장면
- 또는 택배 상자 받는 만족스러운 표정

**핵심 메시지:**
- "마트 가지 마세요" → 페인 포인트 (외출 귀찮음)
- "클릭 한 번" → 편의성
- "미친특가" → 가격 혜택
- "문앞까지" → 배송 편의

## 소재 2: 충격 + 호기심형

**제목:** 이 가격 실화?
**문구:** 눈으로 봐도 믿기지 않는 가격! 미친특가 지금 확인 안 하면 평생 후회합니다
**길이:** 40자
**스타일:** 호기심 자극 + 긴급성
**CTA:** 지금 확인하기
**예상 CTR:** 15-17% (호기심 자극)

**사용 이미지:**
- 가격표 클로즈업 (정가 대비 할인가 강조)
- 또는 놀란 표정의 모델

**핵심 메시지:**
- "눈으로 봐도 믿기지 않는" → 베스트 광고 "눈으로봐도" 벤치마크
- "평생 후회" → 강한 긴급성

## 소재 3: 질문 + 공감형

**제목:** 아직도 정가?
**문구:** 아직도 정가 주고 사세요? 똑똑한 사람들은 이미 미친특가로 반값에 쟁여놨어요
**길이:** 41자
**스타일:** 질문형 + 사회적 증거
**CTA:** 나도 특가로 사기
**예상 CTR:** 13-15%

**사용 이미지:**
- 할인 전/후 가격 비교
- 또는 쇼핑백 든 만족스러운 고객

**핵심 메시지:**
- "아직도 정가?" → 손해 보고 있다는 암시
- "똑똑한 사람들은 이미" → 사회적 증거
- "반값에 쟁여놨어요" → 구체적 혜택

**제목:** 오늘만 이 가격
**문구:** 띠링! 미친특가 알림왔어요. 선착순 한정 수량이라 지금 아니면 내일은 정가입니다
**길이:** 42자
**스타일:** 긴급성 + 희소성
**CTA:** 지금 바로 구매
**예상 CTR:** 14-16%

**사용 이미지:**
- 알림 아이콘 + 타이머
- 또는 재고 떨어지는 애니메이션

**핵심 메시지:**
- "띠링!" → 베스트 광고 "띠링" 벤치마크
- "선착순 한정" → 희소성
- "내일은 정가" → 긴급성

**문구:** 가볍게 보지 마세요. 묵직한 할인에 묵직한 사은품까지! 미친특가는 다릅니다

**핵심 메시지:**

## 💡 사용 팁

### 이미지/영상 제작 가이드

**타입 1: 가격 강조**
- 정가 vs 할인가 명확히 표시
- 큰 폰트, 대비 강한 색상
- "50% OFF" 배지

**타입 2: 타겟 특정**
- 주부: 주방/거실 배경
- 직장인: 사무실/야근 장면
- 실제 타겟과 유사한 모델 사용

**타입 3: 긴급성 강조**
- 카운트다운 타이머
- "한정 수량" 텍스트
- 빨강/주황 색상 사용

**타입 4: 사회적 증거**
- 누적 판매 숫자
- 별점 그래픽
- 실제 고객 후기 캡처

### 동영상 소재 (15초 버전)

**구조:**
- 0-3초: 문제 제기 ("아직도 정가 주고 사세요?")
- 3-7초: 솔루션 ("미친특가 반값!")
- 7-12초: 혜택 ("지금 주문하면 무료배송까지")
- 12-15초: CTA ("지금 바로 확인")

**필수 요소:**
- 무음으로도 이해 가능 (자막 필수)
- 첫 3초에 핵심 메시지
- 모바일 세로 화면 최적화

### 1. 🔴 **배송비 장벽 (가장 큰 원인, 60%)**

**문제:**
```

배송비:     ₩3,500 (23%)
─────────────────────
실제 결제: ₩15,400
```

**왜 문제인가?**
- 배송비가 제품 가격의 **23%**로 과도하게 높음
- 심리적 저항: "11,900원짜리에 배송비 3,500원?"
- 무료배송 기준 ₩50,000 = 제품 4세트 이상 구매 필요
  - 4세트 = ₩47,600 + 배송비 무료 = ₩47,600
  - 하지만 처음 구매자가 4세트를 사기에는 부담

**실제 클릭 후 이탈 시나리오:**
```
1. 광고 클릭 → "11,900원! 좋네!"
2. 장바구니 담기
3. 결제 화면 → "어? 배송비 3,500원?"
4. "15,400원이네... 그냥 마트 가서 살까?"
5. 이탈 (구매 포기)
```

**증거:**
- 39명이 클릭했지만 0명이 구매 = 100% 결제 단계 이탈
- 장바구니 이탈율: 추정 100%

1. **무료배송 프로모션** (단기)

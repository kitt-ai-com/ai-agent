> 결과물을 복사해서 Claude Code에 `knowledge-prompt.md 실행해줘` + 붙여넣기 하면 됩니다.

---

## 프롬프트 (복사용)

```
이 영상을 아래 형식에 맞춰 정리해줘. 빠짐없이, 구조적으로, 개발자가 실무에 바로 쓸 수 있도록.

# 원본 콘텐츠

## 메타 정보
- 소스 타입: 유튜브
- 원본 URL: (영상 URL)
- 제목: (영상 제목)
- 저자/채널: (채널명)
- 날짜: (업로드일)
- 영상 길이: (분:초)

## 타임스탬프별 핵심 정리

| 시간 | 주제 | 핵심 내용 (2-3문장) |
|------|------|---------------------|
| 00:00 | [주제] | [내용] |
| ... | ... | ... |

(영상의 흐름에 따라 주요 전환 지점마다 작성. 최소 10개 이상)

## 전체 내용 정리

영상에서 다루는 모든 내용을 빠짐없이 서술형으로 정리해줘.
단순 요약이 아니라, 영상을 안 봐도 내용을 완전히 이해할 수 있을 정도로 상세하게.
코드가 나오면 코드 블록으로 포함해줘.

### [주제 1]

(상세 설명)

### [주제 2]

(상세 설명)

(영상의 논리 흐름에 따라 섹션을 나눠서)

## 코드 및 실습 요소

영상에서 보여준 코드, 명령어, 설정 파일 등을 모두 정리해줘.

```언어
// 코드 블록
```

- 각 코드의 목적과 맥락 설명 포함
- 실행 방법이 있으면 포함

## 핵심 용어

| 용어 | 설명 |
|------|------|
| [용어1] | [한줄 설명] |
| [용어2] | [한줄 설명] |

## 발표자의 핵심 주장/인사이트

1. (가장 중요한 메시지)
2. (두번째 인사이트)
3. (세번째 인사이트)

> 발표자가 강조한 부분, 반복한 부분, "이게 중요합니다"라고 한 부분 위주로
```

---

## 사용법

1. NotebookLM에 영상 추가 (유튜브 URL 붙여넣기)
2. 위 프롬프트를 채팅창에 붙여넣기
3. 결과가 나오면 전체 복사
4. Claude Code에서:
   ```
   knowledge-prompt.md 실행해줘
   ```
5. 소스 입력 시 NotebookLM 결과를 붙여넣기

> NotebookLM이 **원본 텍스트 추출** (Task #1 collector 역할)을 대신하고,
> Claude Code가 **분석 + 요약 + 실무 적용 + 마인드맵** (Task #2~6)을 처리합니다.

## 1. 타겟 페르소나 정의

### 1-1. 주요 페르소나: "알뜰 미식가 지현씨"

| 항목 | 내용 |
|------|------|
| **이름** | 김지현 (가명) |
| **나이** | 35세 |
| **성별** | 여성 |
| **직업** | 중소기업 마케팅 팀장 (연봉 4,500만 원) |
| **가족 구성** | 맞벌이 부부 + 초등학교 1학년 아들 |
| **거주지** | 경기도 수원시 아파트 |
| **소비 성향** | 가성비 중시, 쿠팡/네이버쇼핑 비교 구매 습관 |

**구체적 고민:**
- 주말마다 아이와 함께할 특별한 식사를 준비하고 싶지만, 수산시장까지 갈 시간이 없다
- 대게를 사주고 싶지만 1인분 3~5만 원대의 가격이 부담스럽다
- 이전에 온라인으로 구매한 해산물이 기대보다 양이 적고 살이 없어 실망한 경험이 있다
- "비품/못난이"라는 표현에 끌리지만, 실제로 먹을 게 있는지 불안하다

**핵심 욕구:**
- 외식 비용의 1/3 이하로 "집에서 대게/홍게 파티" 경험을 재현하고 싶다
- SNS에 올릴 만한 비주얼의 홍게 상차림을 원한다
- "엄마가 이렇게 맛있는 걸 해줬어"라는 아이의 반응을 얻고 싶다

**일상 시나리오:**
> 금요일 저녁, 퇴근 후 소파에 앉아 스마트폰으로 "홍게 택배"를 검색한다. 이번 주말에는 아이에게 특별한 저녁을 해주고 싶다. 대게는 너무 비싸고, 수산시장은 멀다. 스크롤을 내리다 "2kg 15,900원"이라는 가격에 멈춘다. "이 가격에 진짜 괜찮을까?" 반신반의하며 리뷰를 확인한다. 평점 5.0, "살이 꽉 차있다"는 리뷰를 보고 결제 버튼을 누른다.

## 4. 구매 심리 트리거 포인트 6단계

### 전체 구조 개요

| 단계 | 트리거 유형 | 심리 원리 | 배치 위치 (상세페이지 내) | 핵심 메시지 |
|:----:|:----------:|:---------:|:------------------------:|:----------:|
| 1 | **감각 자극** | 본능적 욕구 환기 | 최상단 히어로 영역 (0~10%) | "침 넘어가는 홍게살 클로즈업" |
| 2 | **불안 해소** | 손실 회피 | 상단 1/4 (10~25%) | "정품 수율 90%, 살이 없으면 어쩌지? 걱정 끝" |
| 3 | **사회적 증거** | 동조 효과 / 밴드왜건 | 중상단 (25~40%) | "평점 5.0 / 25개 리뷰 / 재구매율 강조" |
| 4 | **가격 정당화** | 앵커링 + 프레이밍 | 중단 (40~55%) | "대게 1/3 가격, 하루 2,271원" |
| 5 | **긴급성 유발** | 희소성 + FOMO | 중하단 (55~70%) | "제철 시즌 한정 / 물량 소진 시 조기 마감" |
| 6 | **행동 촉구** | 현상 유지 편향 타파 | 하단 CTA (70~100%) | "지금 주문하면 1~3일 내 신선 배송" |

### 각 단계 상세 설계

#### [트리거 1] 감각 자극 — "침이 넘어가게 만들어라"
- **위치:** 상세페이지 진입 즉시 (히어로 이미지)
- **구현:** 김이 모락모락 나는 찐 홍게 클로즈업 / 홍게살을 발라먹는 손 클로즈업
- **심리 원리:** 시각적 자극 → 미각 상상 → 즉각적 욕구 형성
- **카피:** "독도 바다가 키운 꽉 찬 홍게살, 한 입이면 알 수 있습니다"

#### [트리거 2] 불안 해소 — "실패 경험을 치유하라"
- **위치:** 히어로 바로 아래
- **구현:** "온라인 홍게 구매, 이런 걱정하셨죠?" → 걱정 리스트 → 각각에 대한 해소 메시지
- **심리 원리:** 손실 회피(Loss Aversion) — 기존 실패 경험의 반복을 두려워하는 심리 해소
- **카피 구조:**
  - "살이 없으면?" → "정품 수율 90%, 매 마리 살이 꽉 찬 것만 선별"
  - "비린내 나면?" → "독도 청정 해역 당일 출하, 신선도 보장"
  - "양이 적으면?" → "2kg 기준 5~8마리, 3~4인 가족 충분"

#### [트리거 3] 사회적 증거 — "다른 사람들도 만족했다"
- **위치:** 상세페이지 상위 25~40% 구간
- **구현:** 리뷰 하이라이트 캐러셀 + 평점 배지 + 재구매 고객 후기
- **심리 원리:** 사회적 증거(Social Proof) + 밴드왜건 효과
- **핵심 요소:**
  - "평점 5.0 / 25개 리뷰" 큰 배지
  - 대표 리뷰 3개 선정 (가족 식사용, 혼술 안주용, 선물용)
  - "재구매 고객 OO%"가 확인 가능하면 표기

#### [트리거 4] 가격 정당화 — "이건 비싼 게 아니라 싼 거다"
- **위치:** 상세페이지 중단 40~55% 구간
- **구현:** 가격 앵커링 전략(3장)의 4단계 시퀀스 비주얼 적용
- **심리 원리:** 앵커링 효과 + 프레이밍 효과
- **핵심 요소:**
  - 경쟁 가격 비교표 (대게, 마트 홍게, 수산시장)
  - "하루 2,271원" 일일 환산 비용
  - "커피 한 잔 가격으로 홍게 파티" 비교 프레이밍

#### [트리거 5] 긴급성 유발 — "지금 아니면 놓친다"
- **위치:** 상세페이지 중하단 55~70% 구간
- **구현:** 제철 시즌 한정 알림 + 재고 소진 경고
- **심리 원리:** 희소성(Scarcity) + FOMO(Fear of Missing Out)
- **핵심 요소:**
  - "2~3월 제철 홍게 시즌 한정 판매"
  - "오늘 주문 가능 수량: OO개" (재고 표시가 가능한 경우)
  - "이번 시즌 마지막 물량일 수 있습니다"

#### [트리거 6] 행동 촉구 — "지금 바로 누르세요"
- **위치:** 상세페이지 하단 + 플로팅 CTA
- **구현:** 명확한 CTA 버튼 + 배송 정보 + 안심 요소
- **심리 원리:** 현상 유지 편향 타파 + 인지 부하 최소화
- **핵심 요소:**
  - CTA 버튼: "15,900원에 주문하기" (가격 포함 CTA)
  - "무료배송 / 1~3일 내 도착"
  - "교환/반품 7일 이내 가능 — 걱정 없이 주문하세요"
  - 플로팅 하단바: 옵션 선택 + 바로 구매 버튼 상시 노출

## 5. 전환 퍼널 구조 설계 (AIDA+ 변형)

### 5-1. 퍼널 구조: A-I-D-T-A (Attention → Interest → Desire → Trust → Action)

기존 AIDA 모델에 **Trust(신뢰)** 단계를 추가한 변형 모델을 적용합니다.
온라인 수산물 구매는 신선도/품질 불안이 크기 때문에, Desire와 Action 사이에 신뢰 구축 단계가 필수입니다.

```
┌──────────────────────────────────────────────────────────────────┐
│                                                                  │
│   [A] Attention     히어로 이미지 + 한 줄 카피                      │
│         ↓           "독도 홍게 2kg, 단 15,900원"                   │
│                                                                  │
│   [I] Interest      문제 공감 + USP 소개                           │
│         ↓           "대게 가격은 부담, 비품은 불안? 정품 수율 90%"     │
│                                                                  │
│   [D] Desire        상세 혜택 + 가격 앵커링                         │
│         ↓           비주얼 쿡방 + 가격 비교표 + 일일 비용 환산        │
│                                                                  │
│   [T] Trust         리뷰 + 원산지 인증 + 교환/반품 보장              │
│         ↓           "평점 5.0" + "국내산 100%" + "7일 이내 반품"     │
│                                                                  │
│   [A] Action        CTA + 긴급성 + 배송 안내                       │
│                     "지금 주문하면 1~3일 내 무료 배송"               │
│                                                                  │
└──────────────────────────────────────────────────────────────────┘
```

### 5-2. 단계별 상세 설계 및 KPI

| 단계 | 상세페이지 구간 | 핵심 콘텐츠 | 목표 행동 | KPI | 목표 수치 |
|:----:|:--------------:|:----------:|:---------:|:---:|:---------:|
| **Attention** | 0~15% | 히어로 이미지, 핵심 카피, 가격 배지 | 스크롤 시작 | 이탈률(Bounce Rate) | 40% 이하 |
| **Interest** | 15~35% | 고민 공감, USP 3가지, 수율 비교 | 계속 스크롤 | 스크롤 도달률 (35% 지점) | 60% 이상 |
| **Desire** | 35~60% | 조리 비주얼, 가격 앵커링, 옵션 안내 | 옵션 클릭 | 스크롤 도달률 (60% 지점) | 45% 이상 |
| **Trust** | 60~80% | 리뷰, 원산지 인증, 교환/반품 정책 | 불안 해소 | 스크롤 도달률 (80% 지점) | 35% 이상 |
| **Action** | 80~100% + 플로팅 | CTA 버튼, 긴급성, 배송 일정 | 구매 클릭 | CTA 클릭률 | 8~12% |

### 5-3. 퍼널 이탈 방지 장치

| 이탈 지점 | 예상 이탈 사유 | 방지 장치 |
|-----------|---------------|-----------|
| Attention 단계 | "홍게? 별로인데" 관심 부족 | 강렬한 비주얼 + 가격 임팩트("15,900원") 즉시 노출 |
| Interest 단계 | "온라인 해산물은 믿을 수 없어" | 수율 90% 데이터 + 실물 사진으로 즉시 불안 해소 |
| Desire 단계 | "가격이 너무 싼데 괜찮을까?" | 가격 앵커링으로 "싼 이유"가 아니라 "가성비가 좋은 이유" 프레이밍 |
| Trust 단계 | "리뷰가 적어서 불안" | 5.0 만점 강조 + 구체적 리뷰 내용 노출 + 교환/반품 보장 강조 |
| Action 단계 | "나중에 사야지" | 제철 한정 + 재고 소진 경고 + 플로팅 CTA 상시 노출 |

## 7. 비주얼 톤 & 무드 방향 가이드

### 7-1. 전체 톤 & 무드 키워드

> **"신선하고 풍성한 바다의 선물을 합리적으로 즐기는 스마트한 선택"**

| 키워드 | 설명 |
|--------|------|
| **풍성함(Abundance)** | 홍게가 쌓여 있는 이미지, 넘치는 살, 가득 찬 접시 — "양이 충분하다"는 시각적 확신 |
| **신선함(Freshness)** | 바다 색감(딥블루, 청록), 물방울, 얼음 위의 홍게 — 냉장 신선 배송의 시각적 표현 |
| **따뜻함(Warmth)** | 김이 나는 찜 홍게, 가족 식탁, 따뜻한 조명 — 집에서 즐기는 행복한 식사 장면 |
| **합리성(Smart Value)** | 깔끔한 가격 비교 인포그래픽, 숫자 강조 — 가성비를 이성적으로 납득시키는 요소 |

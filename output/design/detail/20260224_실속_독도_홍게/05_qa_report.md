## 1. 항목별 검수 결과

### 1-1. 카피 톤앤매너 일관성 — PASS

| 세부 항목 | 판정 | 비고 |
|----------|------|------|
| 구어체 유지 | PASS | 전 구간에서 "~해요", "~거든요", "~세요" 등 친근한 구어체가 일관되게 유지됨 |
| 브랜드 보이스 통일 | PASS | "과장 없이 수치로 말하는 스타일"이라는 전략 브리프 방향이 카피 전반에 잘 반영됨 |
| 격식 수준 일관성 | PASS | ZONE 1~6 전체에서 존댓말 + 부드러운 권유형 어미 통일 |
| 감정 톤 흐름 | PASS | ZONE 1~2(공감/부정) → ZONE 3(전환/확신) → ZONE 4~5(욕구/긴급) → ZONE 6(안심)으로 자연스러운 감정 아크 구현 |

**피드백:**
- 카피 전반의 톤앤매너가 매우 일관적이며, 전략 브리프에서 정의한 "신뢰감 있는 클린 뷰티 + 과학적 근거" 톤을 잘 살리고 있음
- ZONE 2의 "세럼이 문제였어요"와 ZONE 3의 "네. 가능합니다." 등 짧고 강한 문장으로 리듬감이 좋음
- 다만 ZONE 6 FAQ 답변 중 "순한 편이에요"는 화장품 표현 규제상 주의 필요 (후술)

### 1-2. 전략 → 심리 → 카피 → 비주얼 간 논리적 흐름 연결성 — PASS

| 연결 구간 | 판정 | 비고 |
|----------|------|------|
| 전략 → 심리 | PASS | 전략 브리프의 트리거 포인트 6개가 심리 트리거 맵에서 7단계로 확장되어 심화됨 |
| 심리 → 카피 | PASS | 심리 맵의 각 단계별 "고객 내면 대화"가 카피의 각 ZONE 도입부에 그대로 반영됨 |
| 카피 → 비주얼 | PASS | 카피의 6존 구조를 비주얼 가이드가 1:1로 매핑하여 레이아웃 설계 |
| 전략 USP → 카피 반영 | PASS | 3대 USP(순수 비타민C 15%, 3중 히알루론산, 72시간 보습)가 카피 ZONE 3에 정확히 반영 |
| 가격 앵커링 전략 → 카피 | PASS | 전략 브리프의 3단 앵커 구조(67,000원 → 58,000원 → 39,800원)가 카피 ZONE 4에 동일하게 적용 |
| 퍼널 구조 → 카피 구조 | PASS | 전략의 AIDA+ 변형 퍼널 9단계가 카피 6존 구조에 논리적으로 매핑됨 |

**피드백:**
- 4개 문서 간 핵심 수치(72시간, 15%, 94%, 39,800원 등)가 정확하게 일치함
- 전략 브리프의 페르소나 시나리오("월요일 아침 7시...")가 카피 ZONE 2와 ZONE 4의 시나리오 카피에 자연스럽게 녹아 있음
- 심리 트리거 맵에서 제시한 "감정 곡선"이 카피의 감정 흐름과 비주얼의 톤 변화(밝음→다크→밝음)에 모두 반영되어 삼위일체가 잘 이루어짐

| 구매 여정 단계 | 매핑 ZONE | 판정 | 비고 |
|--------------|----------|------|------|
| Stage 1: 시선 멈춤 (0~3초) | ZONE 1 (히어로) | PASS | 패턴 인터럽트 + 자기 참조 효과 적용 |
| Stage 2: 공감 (3~10초) | ZONE 1~2 (히어로+문제인식) | PASS | 미러링 효과 + 확증 편향 활용 |

| Stage 4: 의심 (30초~1분) | ZONE 3 (솔루션+증거) | PASS | 사회적 증거 + 권위 + Q&A 선제 대응 |

**피드백:**
- 7단계 여정이 6존에 논리적으로 잘 매핑됨 (Stage 2~3이 ZONE 1~3에 걸쳐 자연스럽게 분포)
- 심리 트리거 맵에서 각 단계의 "고객 내면 대화"가 카피 각 존의 도입부에 일관되게 표시되어, 기획자-카피라이터-디자이너 간 의도 정렬이 우수함

### 1-4. 이탈 방지 체크리스트 (8개 항목) — 7/8 PASS

| # | 체크 항목 | 판정 | 근거 |
|---|----------|------|------|
| 1 | 첫 화면에서 3초 안에 타겟의 핵심 고민을 짚고 있는가? | PASS | ZONE 1 헤드라인 "아침마다 당기는 피부, 오후면 무너지는 메이크업" — 타겟 페르소나의 핵심 고민 2가지를 정확히 짚음 |
| 2 | Above the fold에 후킹 요소가 있는가? | PASS | 메인 헤드라인 + 서브 카피("72시간 촉촉함의 비밀, 39,800원") + 모델/제품 이미지 + 태그 3개 + 스크롤 유도 카피가 Above the fold에 배치됨 |
| 3 | 고객이 "나 얘기"라고 느낄 수 있는 구체적 상황이 있는가? | PASS | ZONE 2 페인포인트 체크리스트 5개 항목 모두 구체적 일상 상황 ("아침에 일어나면 볼이랑 턱 라인이 바짝 당긴다", "올리브영 세럼 코너에서..." 등) |
| 4 | 효과 증명 데이터/리뷰가 3개 이상 있는가? | PASS | 체험단 후기 3개 + 통계 데이터 3종(94% 재구매, 4.8 별점, 12,847 판매) + 전문가 코멘트 1개 = 총 7개 증거 |
| 5 | 경쟁 제품 대비 왜 이 제품인지 명확한가? | PASS | ZONE 4 비교표에서 경쟁 A(고가) / 경쟁 B(저가) 대비 성분 우위 + 가격 경쟁력을 5개 항목으로 시각적 비교 |
| 6 | "지금 사야 하는 이유"가 구체적인가? | PASS | 카운트다운 타이머 + 한정 3,000개 잔여 수량 + "놓치면 18,200원 추가 지불" + 선착순 500명 미니 사이즈 증정 — 4중 긴급성 트리거 |
| 7 | 환불/교환 안전망이 눈에 띄는가? | PASS | ZONE 6에서 방패 아이콘 + "14일 100% 환불 보장, 반송비 무료" + 환불 요청률 0.8% 공개 + 안심 체크리스트 5개 항목 |
| 8 | CTA 버튼이 2~3회 노출되는가? | PARTIAL PASS | ZONE 5에서 1회("런칭 특가로 시작하기"), ZONE 6에서 1회("부담 없이 시작하기")로 총 2회 노출. 최소 기준은 충족하나, ZONE 3 하단이나 ZONE 4 하단에 1회 추가 노출 권장 |

**피드백:**
- 8개 항목 중 7개 완전 통과, 1개 조건부 통과
- CTA 버튼 노출 횟수가 2회로 최소 기준은 충족하지만, 스크롤이 긴 상세페이지 특성상 ZONE 3(솔루션+증거) 하단 또는 ZONE 4(가격 비교) 하단에 중간 CTA를 1회 추가하여 총 3회 노출을 권장함

### 1-6. 법적 표기사항 누락 여부 (화장품 관련) — FAIL

| 법적 필수 표기 항목 | 판정 | 비고 |
|-------------------|------|------|
| 전성분 표기 | FAIL | 심리 트리거 맵에서 "전성분 표기"를 투명성 시그널로 언급했으나, 카피 원고와 비주얼 가이드에 실제 전성분 표기 섹션이 없음 |
| 제품명/용량/가격 표시 | PARTIAL | 제품명과 가격은 표기되어 있으나, 용량(ml) 표기가 카피 본문에 명시적으로 없음 (비주얼 가이드 ZONE 5에 "30ml" 언급만 프롬프트에 존재) |
| 제조업자/책임판매업자 | FAIL | 표기 없음 |
| 원산지/제조국 | FAIL | 표기 없음 |
| 사용기한/개봉 후 사용기한 | FAIL | 표기 없음 |
| 사용 시 주의사항 | FAIL | "목 뒤에 먼저 테스트해보세요"라는 FAQ 답변이 있으나, 법적 필수 주의사항 형식이 아님 |
| 기능성 화장품 심사 여부 | FAIL | "브라이트닝", "톤 개선" 등 미백 효능을 암시하는 표현이 있으나, 기능성 화장품 인증 표기 없음 |
| "72시간 보습" 클레임 근거 | FAIL | 구체적 임상 시험 결과 또는 인체적용시험 기관/결과 표기 없음 |
| 광고 표현 적법성 | WARNING | "민감성 피부 92% 무자극 확인" — 피험자 수, 시험 기관명 미기재 |

**피드백 (중요도 높음):**

이 항목은 상세페이지 라이브 전 반드시 보완해야 합니다.

1. **전성분 표기**: 화장품법 시행규칙에 따라 상세페이지 내 전성분 표기 필수. ZONE 6 하단 또는 별도 섹션에 전성분 리스트 추가 필요
2. **제품 용량 표기**: 카피 본문 내 "30ml" (또는 실제 용량) 명시 필요
3. **제조업자/책임판매업자/원산지**: ZONE 6 하단에 법적 필수 정보 블록 추가 필요
4. **사용기한**: "제조일로부터 OO개월" 또는 "개봉 후 OO개월 이내 사용" 표기 필요
5. **기능성 화장품 심사**: "톤 개선", "브라이트닝", "잡티도 옅어진 것 같아서" 등의 표현이 미백 효능을 암시 → 기능성 화장품 심사를 받지 않았다면 해당 표현 삭제 또는 수정 필요
6. **임상 근거 표기**: "72시간 보습", "민감성 피부 92% 무자극" 등 구체적 수치 클레임에는 "OO기관, 피험자 OO명, OOOO년 OO월 시험 결과" 형태의 근거 표기 필수
7. **"피부과 전문의 코멘트"**: 실제 전문의의 성명과 소속 기관을 명시해야 하며, 허위 전문가 코멘트 사용 시 법적 문제 발생 가능

### 1-7. 경쟁사 대비 차별화 포인트 명확성 — PASS

| 차별화 항목 | 판정 | 비고 |
|-----------|------|------|
| 성분 차별화 | PASS | 순수 비타민C 15%(유도체 대비 흡수율 우위) + 3중 히알루론산(단일 분자 대비 3개 층 침투) — 과학적 근거로 차별화 |
| 보습 지속 차별화 | PASS | 72시간(경쟁 A: 48시간, 경쟁 B: 24시간) — 3배 차이를 시각적 타임라인으로 명확히 표현 |
| 가격 차별화 | PASS | 고가 브랜드 대비 40% 저렴하면서 성분은 동급 이상 — "가성비 최강" 포지셔닝 명확 |
| 안전성 차별화 | PASS | 피부 자극 테스트 완료 + 무향료/무색소 — 경쟁 제품 중 일부만/미실시 대비 확실한 우위 |
| 비교표 시각화 | PASS | ZONE 4에서 3열 비교표로 한눈에 차이 파악 가능, 글로우핏 열에 오렌지 강조 + BEST 배지 |

**피드백:**
- 차별화 포인트가 "성분 > 효능 > 가격 > 안전성" 4가지 축에서 모두 경쟁 우위를 주장하고 있어 설득력 높음
- 비교 대상을 "고가"(성분 대등, 가격 저렴)와 "저가"(가격 유사, 성분 우위)로 양쪽에 배치하여 글로우핏이 중간에서 최적 포지션을 차지하는 "가운데 배치 효과" 활용이 전략적으로 우수함
- 단, 경쟁사 실명 미사용("백화점 A 세럼", "로드샵 B 세럼")으로 비교광고 규정은 준수하고 있음

## 3. A/B 테스트 변수 설계 3가지

### 테스트 A: 히어로 헤드라인 (ZONE 1) — 감정형 vs 수치형

### 테스트 C: 신뢰 요소 배치 순서 (ZONE 3) — 후기 먼저 vs 성분 먼저

| 항목 | 변형 A (현재: 성분 먼저) | 변형 B (후기 먼저) |
|------|----------------------|-------------------|

| **접근법** | 논리적 설득(성분/과학) → 사회적 증거(후기) 순서 | 사회적 증거(후기) 먼저 → 논리적 뒷받침(성분) 순서 |
| **가설** | 성분에 민감한 타겟에게는 성분 먼저가 효과적이나, 리뷰를 가장 중시하는 고객에게는 후기 먼저가 스크롤 깊이를 높일 수 있음 |
| **측정 지표** | 스크롤 깊이 50% 도달률, ZONE 3 체류 시간, 스크롤 깊이 75% 도달률 |

| **최소 표본** | 각 변형당 1,500 세션 이상 |

**A/B 테스트 우선순위:** A(히어로 헤드라인) > C(신뢰 요소 배치) > B(가격 표시)
- 테스트 A를 가장 먼저 실행하는 이유: 상단 이탈률이 전체 전환에 가장 큰 영향을 미치므로, 히어로 영역 최적화가 선행되어야 함
- 테스트 A 결과 확정 후 B와 C를 동시 진행 가능 (서로 다른 ZONE이므로 상호 간섭 없음)

## 2. 경쟁 상품 대비 USP 도출

### 경쟁 비교표

| 비교 항목 | 우리 상품 | 경쟁사 A (냉동 LA갈비) | 경쟁사 B (마트 LA갈비) | 경쟁사 C (한우 갈비) |
|-----------|-----------|----------------------|---------------------|-------------------|
| **가격/kg** | **27,500원/kg** | 30,000~35,000원/kg | 35,000~40,000원/kg | 60,000원 이상/kg |
| **상태** | **생물 (신선)** | 냉동 | 냉장 (진열 기간 불명) | 냉장/냉동 혼재 |
| **비계 비율** | **최소 (뼈+살코기 only)** | 비계 포함 중량 | 비계 포함 중량 | 비계 적으나 가격 높음 |
| **실중량** | **2kg 순살+뼈** | 2kg 표기(비계 포함) | 소분 500g 단위 | 500g~1kg |
| **배송비** | **무료** | 3,000~5,000원 | 직접 구매 | 무료(고가) |
| **배송 속도** | **당일발송(9시 전)** | 2~3일 | 즉시 | 1~2일 |
| **평점** | **5.0/5** | 4.2~4.5 | 확인 불가 | 4.5~4.8 |

### 핵심 USP 3가지

> **USP 1 (Primary): "비계 없는 실중량 2kg"**
> - 경쟁사 대부분 비계 포함 중량 표기 → 실제 먹을 수 있는 살코기는 적음
> - 우리 제품은 "뼈와 살코기 무게로만 2kg" → 실속 차별화
> - 소비자 언어: "비계에 돈 내지 마세요. 진짜 살코기 2kg입니다"

> **USP 2: "생물(냉동X) 신선 배송"**
> - 냉동 갈비 해동 시 육즙 손실, 식감 저하 문제 해결
> - 받자마자 바로 조리 가능 → 편의성 + 맛 품질 동시 확보
> - 소비자 언어: "얼리지 않았습니다. 정육점 그 신선함 그대로"

> **USP 3: "kg당 27,500원의 압도적 가성비"**
> - 마트 대비 30% 이상 저렴, 한우 대비 50% 이상 저렴
> - 무료배송까지 포함하면 실질 가격 격차 더 확대
> - 소비자 언어: "마트 1kg 값으로 여기선 2kg 삽니다"

## 3. 가격 앵커링 전략

### 4단계 가격 프레이밍

```
[1단계] 높은 앵커 설정
"정육점/마트 LA갈비 평균 kg당 35,000~40,000원"
→ 2kg 기준 70,000~80,000원이 '보통 가격'이라는 인식 심기

[2단계] 정가 취소선
"정가 65,000원" ← 취소선 처리
→ 이미 마트보다 저렴한 가격도 '정가'였다는 인식

[3단계] 할인가 강조
"오늘 특가 55,000원 (15% OFF, 10,000원 절약!)"
→ 이중 할인 효과: 마트 대비 절약 + 정가 대비 절약

[4단계] 일일 비용 환산 (Pain Minimizing)
"하루 1,833원으로 온 가족 LA갈비 파티!"
→ 55,000원 ÷ 30일 = 1,833원/일
→ 또는: "1인분 13,750원, 외식 갈비 1인분(25,000~35,000원)의 절반!"
→ 55,000원 ÷ 4인분 = 13,750원
```

### 추가 앵커링 장치

- **비계 포함 제품 대비 실질 가격**: "비계 30% 포함된 타사 2kg = 실살코기 1.4kg → kg당 실질 46,400원 vs 우리 27,500원"
- **외식 비교**: "갈비집 2인 세트 8~10만원 vs 4인 가족 실컷 55,000원"
- **무료배송 강조**: "배송비 0원 → 실질 할인 추가 3,000~5,000원"

## 4. 전환 퍼널 구조 (AIDA+ 변형)

### 6-ZONE 퍼널 모델

```
ZONE 1: STOP (시선 정지)          ← Attention
  ↓ 3초 내 스크롤 멈춤
ZONE 2: EMPATHY (공감 진입)       ← Interest (감정)
  ↓ "이거 내 얘기잖아"
ZONE 3: PROOF (증거 제시)         ← Interest (논리) + Desire
  ↓ "진짜네, 믿을 만하다"
ZONE 4: DESIRE (욕구 폭발)        ← Desire 극대화
  ↓ "나도 저렇게 먹고 싶다"
ZONE 5: OVERCOME (불안 해소)      ← 구매 장벽 제거
  ↓ "살 이유 > 안 살 이유"
ZONE 6: ACTION (긴급 행동)        ← Action + Urgency
  ↓ "지금 안 사면 후회한다"
  [구매 버튼 CTA]
```

### 각 ZONE 목표 & 핵심 메시지

| ZONE | 목표 | 체류 시간 | 핵심 메시지 |
|------|------|-----------|-------------|
| 1. STOP | 스크롤 정지 | 0~3초 | 충격적 비주얼 + 한 줄 후킹 카피 |
| 2. EMPATHY | 감정 연결 | 3~8초 | "비계에 돈 내던 경험" 공감 |
| 3. PROOF | 신뢰 구축 | 8~20초 | 실물 비교 사진 + 리뷰 + 수치 |
| 4. DESIRE | 구매 욕구 | 20~40초 | 조리 장면 + 가족/캠핑 시나리오 |
| 5. OVERCOME | 장벽 제거 | 40~60초 | FAQ + 보증 + 배송 안내 |
| 6. ACTION | 즉시 구매 | 60초~ | 한정 수량 + CTA + 추가 혜택 |

## 1. 고객 구매 여정 7단계: 내면 대화 + 넘겨야 할 벽

### STAGE 1: 시선멈춤 (Scroll-Stop)

**내면 대화:**
> "어? 이거 뭐지... LA갈비가 55,000원에 2kg? 잠깐만..."

**넘겨야 할 벽:** 수천 개 상품 속 무관심의 벽
- 극복 전략: 시각적 충격(붉은 생고기 클로즈업) + 숫자 강조("2kg / 55,000원") + 의외성 카피
- 핵심 원리: **초두효과(Primacy Effect)** + **폰 레스토프 효과(고립 효과)**

---

### STAGE 2: 공감 (Empathy)

**내면 대화:**
> "맞아... 지난번에 산 갈비도 비계가 반이었어. 돈 아까웠는데... 이 사람이 내 마음을 아네?"

**넘겨야 할 벽:** "이건 남의 얘기야"라는 거리감
- 극복 전략: 구체적 불편 경험 묘사 → 타겟의 기억 속 실패 경험을 활성화
- 핵심 원리: **미러링(Mirroring)** + **확증 편향(Confirmation Bias)** 활용

---

### STAGE 3: 관심 (Interest)

**내면 대화:**
> "비계 없이 뼈와 살코기만? 생물이라 신선하다고? 좀 더 자세히 봐볼까..."

**넘겨야 할 벽:** "뻔한 광고겠지"라는 회의감
- 극복 전략: 실물 사진 + 구체적 수치(비계 제거 과정, 실중량 인증) + 간결한 차별점 요약
- 핵심 원리: **정교화 가능성 모델(ELM) 중심 경로** - 논리적 근거로 관심 유지

---

### STAGE 4: 의심 (Skepticism)

**내면 대화:**
> "근데 멕시코산이잖아... 품질 괜찮을까? 55,000원이면 싼데 뭔가 있는 거 아냐? 리뷰도 8개밖에 없네..."

**넘겨야 할 벽:** 가격-품질 의심의 벽 (저렴 = 저품질 연상)
- 극복 전략:
  - 멕시코산 LA갈비의 우수성 설명 (미국산과 동일 품종, USDA 기준 사육)
  - 가격이 싼 이유를 투명하게 설명 (유통 단계 축소, 자체 수입)
  - 리뷰 8개 전부가 5점 만점이라는 점 강조 (양보다 질)
  - 실구매자 포토리뷰 배치
- 핵심 원리: **사회적 증거(Social Proof)** + **투명성 효과(Transparency Effect)**

---

### STAGE 5: 욕구 (Desire)

**내면 대화:**
> "와... 저 구워진 갈비 사진 보니까 침 고인다. 주말에 가족이랑 이거 구워 먹으면 아이들이 진짜 좋아하겠다!"

**넘겨야 할 벽:** 이성적 판단에서 감성적 욕구로의 전환
- 극복 전략: 조리 완성 비주얼 + 사용 시나리오(가족 식탁, 캠핑장) + 오감 자극 카피
- 핵심 원리: **심상 처리(Mental Imagery)** + **소유 효과 예상(Endowment Effect Anticipation)**

---

### STAGE 6: 망설임 (Hesitation)

**내면 대화:**
> "근데 2kg면 너무 많지 않나? 다 못 먹으면 어쩌지... 배송 중에 상할 수도 있잖아... 다음에 살까?"

**넘겨야 할 벽:** 실패 불안 + 현상유지 편향
- 극복 전략:
  - 2kg 활용법 제시 (1kg은 당일 조리, 1kg은 냉동 보관)
  - 아이스박스 + 냉장 특수 포장 강조
  - "오늘 안 사면 다음번엔 가격이 오를 수 있습니다" (손실 회피)
  - 교환/반품 7일 보장
- 핵심 원리: **손실 회피(Loss Aversion)** + **현상유지 편향 깨기**

---

### STAGE 7: 불안해소 (Reassurance)

**내면 대화:**
> "반품도 되고, 무료배송이고, 리뷰도 다 좋고... 이 가격이면 안 사는 게 손해지. 장바구니 담자!"

**넘겨야 할 벽:** 결제 직전의 마지막 저항
- 극복 전략:
  - 안전장치 총정리 (무료배송 + 7일 반품 + 불량 시 배송비 무료)
  - 후기 작성 추가 혜택 (포토 500P / 일반 100P)
  - 최종 가격 재확인 + CTA 버튼
- 핵심 원리: **제로 리스크 편향(Zero-Risk Bias)** + **확실성 효과(Certainty Effect)**

## 2. 6개 ZONE별 심리 트리거 배치 맵

### ZONE 1: STOP (시선 정지)

| 항목 | 내용 |
|------|------|
| **목적** | 3초 내 스크롤 멈춤, 무의식적 시선 고정 |
| **핵심 심리 원칙** | **폰 레스토프 효과**(주변과 다른 자극에 시선 집중) + **초두효과**(첫 인상이 전체 판단 지배) |
| **실행안** | |
| 비주얼 | 화면 가득 채운 생LA갈비 클로즈업 (붉은 살코기 + 하얀 뼈의 대비) |
| 카피 | 숫자 중심의 충격적 한 줄 (예: "비계 빼고 2kg, 55,000원") |
| 레이아웃 | 상단 풀블리드 이미지 + 최소 텍스트 (정보 과부하 방지) |
| 추가 장치 | 가격 취소선(~~65,000원~~) → 할인가 강조 배지 |

---

### ZONE 2: EMPATHY (공감 진입)

| 항목 | 내용 |
|------|------|
| **목적** | "이건 내 얘기다"라는 감정적 연결 형성 |
| **핵심 심리 원칙** | **바넘 효과**(자신에게 해당된다고 느끼는 일반적 진술) + **부정성 편향**(부정 경험이 더 강하게 기억됨) + **미러링** |
| **실행안** | |
| 카피 구조 | 문제 제기형 질문 → 공감 서술 → 해결 암시 |
| 예시 카피 | "갈비 샀는데 반이 비계였던 적, 있으시죠?", "냉동 갈비 해동했더니 핏물에 육즙까지 다 빠진 경험..." |
| 비주얼 | 비계 많은 갈비 사진 (실망스러운 경험의 시각화) → 취소선/X 표시 |
| 톤 | 공격적이지 않은, 다정한 공감 톤 ("저희도 그래서 만들었습니다") |

---

### ZONE 3: PROOF (증거 제시)

| 항목 | 내용 |
|------|------|
| **목적** | 감정에서 논리로 전환, "믿을 만하다"는 판단 유도 |
| **핵심 심리 원칙** | **사회적 증거(치알디니)** + **권위의 원칙** + **앵커링 효과** |
| **실행안** | |
| 비교 이미지 | 타사 비계 포함 갈비 vs 우리 살코기 갈비 나란히 비교 (Before/After 구도) |
| 수치 증거 | "비계 제거 후 순살+뼈 2kg → 실중량 표기", kg당 가격 비교표 |
| 리뷰 증거 | 5.0 만점 + 포토리뷰 3~4개 하이라이트 (핵심 문구 볼드 처리) |
| 신뢰 배지 | "HACCP 인증", "당일 도축 → 당일 발송", 원산지 증명서 |
| 가격 앵커링 | 마트 가격 → 정가 취소선 → 할인가 → 1인분 단가 순서로 프레이밍 |

---

### ZONE 4: DESIRE (욕구 폭발)

| 항목 | 내용 |
|------|------|
| **목적** | 이성적 확인 → 감성적 욕구 폭발, "먹고 싶다!" 감정 극대화 |
| **핵심 심리 원칙** | **심상 처리(Mental Imagery)** + **소유 효과(Endowment Effect)** + **호혜의 원칙(치알디니)** - 레시피 제공으로 빚진 감정 |
| **실행안** | |
| 비주얼 | 그릴 위 지글지글 굽히는 영상/GIF + 완성된 갈비 플레이팅 |
| 시나리오 1 | "주말 저녁, 가족 4명이 둘러앉아 갓 구운 LA갈비를 한 점씩..." (가정 시나리오) |
| 시나리오 2 | "캠핑장 숯불 위 LA갈비, 친구들의 감탄사가 절로..." (캠핑 시나리오) |
| 오감 카피 | 시각(선홍빛 마블링) + 청각(지글지글) + 후각(숯불 향) + 미각(달콤한 양념) + 촉각(부드러운 식감) |
| 추가 가치 | 간단 레시피 카드 제공 (양념 갈비 / 소금구이 2가지) |

---

### ZONE 5: OVERCOME (불안 해소)

| 항목 | 내용 |
|------|------|
| **목적** | 모든 구매 장벽 제거, "안 살 이유가 없다"로 전환 |
| **핵심 심리 원칙** | **제로 리스크 편향** + **확실성 효과** + **인지적 종결 욕구(Need for Closure)** |
| **실행안** | |
| FAQ 형식 | Q&A로 예상 불안을 선제 해소 |
| Q1 | "멕시코산 품질 괜찮나요?" → "미국과 동일 환경에서 사육, USDA 품질 기준 충족" |
| Q2 | "2kg 다 못 먹으면?" → "1kg 단위 소분 포장, 남는 건 냉동 보관 가능" |
| Q3 | "배송 중 상하지 않나요?" → "아이스박스 + 아이스팩 냉장 포장, 신선도 보장" |
| Q4 | "마음에 안 들면?" → "배송 후 7일 내 교환/반품, 불량 시 배송비 무료" |
| 보증 배지 | 무료배송 / 당일발송 / 7일 반품 / 신선도 보장 아이콘 4종 가로 배치 |

### ZONE 6: ACTION (긴급 행동 유도)

| 항목 | 내용 |
|------|------|
| **목적** | 즉시 구매 행동 유발, "지금 사야 한다"는 긴급성 부여 |
| **핵심 심리 원칙** | **희소성의 원칙(치알디니)** + **손실 회피(Loss Aversion)** + **행동 관성(Momentum)** + **좋아함의 원칙(치알디니)** |
| **실행안** | |
| 긴급성 카피 | "오늘 주문 마감까지 N시간 N분" (카운트다운 느낌의 텍스트) |
| 손실 프레이밍 | "이 가격은 한정 수량 소진 시 종료됩니다" |
| 가격 최종 정리 | ~~65,000원~~ → **55,000원** + 무료배송 + 후기 작성 시 최대 500P |
| CTA 버튼 | "55,000원에 2kg 받아보기" (금액+혜택 포함 CTA) |
| 추가 넛지 | "지금 N명이 이 상품을 보고 있습니다" (사회적 증거 + 희소성 결합) |
| 스티키 CTA | 스크롤 하단 고정 구매 버튼 (이탈 방지) |

## 3. 감정 곡선 설계

### 전체 감정 흐름 그래프

```
감정 강도
(+) 높음
  |
  |                                          *** 욕구 절정
  |                                       ***   ***
  |                                     **         **
  |                                   **             *  긴급
  |                         신뢰    **                 **
  |                       **  **  **                     ***
  |                     **      **                         *** 안심/확신
  |                   **                                      ***
  |   시선멈춤      **                                           *** → 구매!
  |     **        **
  |   **   **   **
  |  *       * *
  | *         *
  |*        공감(부정)
  |        *
(-) 낮음  *
  +-------|---------|---------|---------|---------|---------|--------→ 스크롤
       ZONE 1    ZONE 2    ZONE 3    ZONE 4    ZONE 5    ZONE 6
       STOP     EMPATHY    PROOF    DESIRE   OVERCOME   ACTION
```

### 5단계 감정 전환 설계

| 단계 | 감정 상태 | 강도 | ZONE | 전환 트리거 |
|------|-----------|------|------|-------------|
| **1. 부정** | 불편함, 과거 실망 회상 | 하 (-) | ZONE 2 | "비계 갈비에 속았던 기억" → 불쾌한 감정 활성화 |
| **2. 희망** | "이런 게 있었어?" 기대감 | 중 (+) | ZONE 2~3 | 문제 인식 → 해결책 제시 전환점 |
| **3. 욕구** | 먹고 싶다, 갖고 싶다 | 최고 (+++) | ZONE 4 | 조리 비주얼 + 오감 자극 → 감각적 욕구 폭발 |
| **4. 긴급** | "놓치면 안 돼" 초조함 | 고 (++) | ZONE 6 | 한정 수량 + 마감 시간 → 구매 행동 압력 |
| **5. 안심** | 안전하다, 후회 없다 | 중상 (+) | ZONE 5~6 | 반품 보장 + 만점 리뷰 → 불안 제거 후 안정적 구매 결정 |

### 감정 전환 핵심 포인트

1. **부정 → 희망 (ZONE 2→3)**: 가장 중요한 전환점. 부정적 감정이 충분히 활성화된 후 "하지만..."으로 전환해야 카타르시스 극대화
2. **희망 → 욕구 (ZONE 3→4)**: 논리적 신뢰가 쌓인 상태에서 감성 자극을 터뜨려야 효과적. 순서가 바뀌면 "광고 같다"는 저항 발생
3. **욕구 → 긴급 (ZONE 4→6)**: 욕구가 최고점일 때 "지금이 아니면 안 된다"는 시간 압박을 추가해야 행동으로 연결
4. **긴급 → 안심 (ZONE 6)**: 마지막에 안전장치를 보여줘서 "충동구매 아니야, 합리적 선택이야"라는 자기 합리화 근거 제공

## 부록: 치알디니 6원칙 적용 맵 요약

| 원칙 | 적용 위치 | 구체적 실행 |
|------|-----------|-------------|
| **상호성(Reciprocity)** | ZONE 4 | 무료 레시피 카드 제공 → 빚진 감정으로 구매 유도 |
| **일관성(Commitment)** | ZONE 3~4 | "좋은 고기를 알아보는 눈"이라는 정체성 부여 → 구매가 일관된 행동 |
| **사회적 증거(Social Proof)** | ZONE 3, 6 | 리뷰 만점 + "N명이 보는 중" + 포토 후기 |
| **권위(Authority)** | ZONE 3 | HACCP 인증, 원산지 증명, 전문 정육 과정 설명 |
| **좋아함(Liking)** | ZONE 2, 4 | 공감 카피로 친근감 형성 + 행복한 식탁 시나리오 |
| **희소성(Scarcity)** | ZONE 6 | 한정 수량 + 시간 제한 + "이 가격은 곧 종료" |

### 추가 심리 원칙 적용

| 원칙 | 적용 위치 | 구체적 실행 |
|------|-----------|-------------|
| **손실 회피** | ZONE 5~6 | "이 가격을 놓치면 10,000원 더 내야 합니다" |
| **앵커링** | ZONE 1, 3 | 마트 가격 → 정가 → 할인가 순차적 프레이밍 |
| **프레이밍 효과** | ZONE 3 | "비계 포함 2kg" vs "순살코기 2kg" 차이 강조 |

| **디폴트 효과** | ZONE 6 | 1개 기본 선택 + "2개 구매 시 추가 할인" 옵션 |

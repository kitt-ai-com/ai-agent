## 2. 경쟁 상품 대비 USP 도출

### 핵심 USP 3가지

**USP 1: "국내산 한돈인데 9,500원" (가격 파괴)**
- 국내산 냉장육의 품질 + 수입 냉동육 수준의 가격
- 경쟁 불가능한 가격 포지셔닝 = "이 가격에 국내산 냉장이 말이 돼?"라는 놀라움

**USP 2: "4종 부위 골라담기" (선택의 자유)**
- 삼겹살, 오겹살, 목살, 항정살 중 원하는 부위 선택
- 구이용/수육용 용도까지 선택 = 내 입맛과 상황에 딱 맞춤
- 경쟁사는 부위별로 따로 사야 하는데, 여기선 한 번에 해결

**USP 3: "냉장 신선육" (품질 보증)**
- 냉동이 아닌 냉장 = 해동 필요 없이 바로 조리 가능
- 냉장육 특유의 육즙과 식감 보존
- "받자마자 바로 구워 먹는" 신선한 경험

### USP 한 줄 요약 (핵심 메시지)
> **"국내산 한돈 냉장육 4종을 9,500원에 골라담는다 — 이건 마트에서도 못 하는 가격이다"**

## 3. 가격 앵커링 전략

### 4단계 가격 인식 설계

```
[1단계: 높은 앵커 설정]
"마트에서 국내산 삼겹살 100g에 1,800~2,500원"
"외식으로 삼겹살 1인분 먹으면 15,000~18,000원"
→ 소비자 머릿속에 "국내산 돼지고기 = 비싸다" 기준선 설정

[2단계: 정가 취소선]
정가 15,500원 ← 취소선 처리
→ "원래 이 정도 가격이 정상이지" 확인 심리

[3단계: 할인가 강조]
→ 9,500원 (39% OFF)
→ "6,000원이나 아끼네!" 이득 프레이밍
→ 39% 할인율 강조로 "거의 반값" 인식

[4단계: 일일 비용 환산]
"하루에 약 317원으로 온 가족이 국내산 한돈을" (한 달 기준)
"편의점 삼각김밥 하나 값도 안 되는 가격"
"배달 치킨 1마리 값으로 3번 이상 구워 먹는다"
→ 저렴함의 체감 극대화
```

### 추가 앵커링 장치
- **50,000원 무료배송 기준** → "5팩만 사면 무료배송 + 한끼당 더 저렴" → 묶음 구매 유도
- **세트 옵션** → 단품 대비 세트 할인 → "세트가 더 이득이네" 상향 구매 유도

## 4. 전환 퍼널 구조 (AIDA+ 변형)

### 6-ZONE 퍼널 (AIDA + Trust + Urgency)

```
┌─────────────────────────────────────────────┐
│  ZONE 1: ATTENTION (시선 멈춤)              │
│  "국내산 한돈이 9,500원?" 충격 후킹          │
│  → 스크롤 멈추게 하는 1.5초 승부             │
├─────────────────────────────────────────────┤
│  ZONE 2: INTEREST (관심 유발)               │
│  4종 부위 비주얼 + 골라담기 재미              │
│  → "이 가격에 이 퀄리티?" 기대감 증폭        │
├─────────────────────────────────────────────┤
│  ZONE 3: DESIRE (욕구 자극)                 │
│  시즐컷 + 먹는 장면 + 일상 시나리오          │
│  → "지금 당장 구워 먹고 싶다" 감각 자극       │
├─────────────────────────────────────────────┤
│  ZONE 4: TRUST (신뢰 구축)                  │
│  리뷰, 원산지, 품질 보증, 반품 안내           │
│  → "사도 괜찮겠다" 의심 해소                 │
├─────────────────────────────────────────────┤
│  ZONE 5: URGENCY (긴급성)                   │
│  한정 수량/기간, 가격 인상 예고               │
│  → "지금 안 사면 손해" 행동 촉발             │
├─────────────────────────────────────────────┤
│  ZONE 6: ACTION (구매 전환)                 │
│  CTA + 무료배송 안내 + 최종 안심             │
│  → "지금 바로 담기" 클릭                     │
└─────────────────────────────────────────────┘
```

### 각 ZONE별 핵심 지표

| ZONE | 목표 | 핵심 KPI | 체류 시간 목표 |
|------|------|----------|---------------|
| 1. Attention | 스크롤 멈춤 | 이탈률 감소 | 1~2초 |
| 2. Interest | 상세 진입 | 스크롤 깊이 30%+ | 5~10초 |
| 3. Desire | 감정 자극 | 스크롤 깊이 60%+ | 10~15초 |
| 4. Trust | 의심 해소 | 리뷰 클릭률 | 5~10초 |
| 5. Urgency | 긴급감 | 장바구니 담기율 | 3~5초 |
| 6. Action | 구매 전환 | 결제 완료율 | 즉시 |

### 비주얼 무드 키워드
- **"숯불 위의 프리미엄"** — 고급스럽지만 거부감 없는 따뜻함
- 그릴 위에서 지글거리는 시즐컷 (연기, 육즙, 기름 방울)
- 어두운 배경(숯/화로) 위 밝게 빛나는 고기 = 주인공 부각
- 가정집 식탁 / 캠핑장 그릴 등 실제 사용 장면 연출
- 4종 부위를 한 눈에 보여주는 그리드 레이아웃

### 폰트 방향
- **헤드라인**: 굵고 강한 고딕체 (예: Pretendard ExtraBold, 나눔스퀘어 ExtraBold)
- **본문**: 가독성 좋은 본문 고딕 (Pretendard Regular)
- **가격/할인**: 숫자 강조용 볼드 + 레드 컬러

### 이미지 스타일 가이드
- 고기 촬영: 45도 각도, 자연광 또는 따뜻한 조명, 배경 아웃포커싱
- 시즐컷: 연기와 기름 방울이 보이는 동적인 순간 포착
- 라이프스타일: 가족 식탁, 캠핑장, 친구 모임 등 "먹는 순간의 행복"
- 원재료컷: 마블링 클로즈업, 신선한 핑크빛 육색 강조

## 1. 고객 구매 여정 7단계별 내면 대화 + 넘겨야 할 벽

### Stage 1: 시선멈춤 (Scroll Stop)

| 항목 | 내용 |
|------|------|
| **고객 내면 대화** | "어? 국내산 한돈이 9,500원? 잠깐, 이거 실화야?" |
| **감정 상태** | 무심 → 놀라움 (0.5초 전환) |
| **넘겨야 할 벽** | **무관심의 벽** — 수백 개 상품 중 왜 이걸 봐야 하지? |
| **벽 넘기 전략** | 가격 충격 + 시즐 비주얼의 동시 타격. "국내산 한돈"과 "9,500원"의 인지 부조화가 스크롤을 멈추게 함 |
| **핵심 심리 원칙** | 인지 부조화(Cognitive Dissonance) — "국내산 = 비싸다"는 기존 믿음과 충돌하는 가격이 호기심 유발 |

### Stage 2: 공감 (Empathy)

| 항목 | 내용 |
|------|------|
| **고객 내면 대화** | "맞아, 요즘 장볼 때마다 고기값 때문에 한숨 나오는데... 국내산은 엄두도 못 냈어" |
| **감정 상태** | 놀라움 → 공감 + 약간의 설렘 |
| **넘겨야 할 벽** | **무관련의 벽** — 이게 나한테 해당되는 이야기인가? |
| **벽 넘기 전략** | 타겟이 공감할 수 있는 일상 고민 터치. "고기값 걱정 없이 온 가족이 배불리", "마트 갈 시간 없는 당신을 위해" |
| **핵심 심리 원칙** | 미러링(Mirroring) — 고객의 현재 불만과 고민을 정확히 비추면 "이건 나를 위한 상품"이라는 인식 형성 |

### Stage 3: 관심 (Interest)

| 항목 | 내용 |
|------|------|
| **고객 내면 대화** | "4종이나 고를 수 있다고? 삼겹살이랑 항정살 같이 시킬 수 있네. 구이용 따로 수육용 따로도 되고..." |
| **감정 상태** | 공감 → 탐색 욕구 |
| **넘겨야 할 벽** | **차별화의 벽** — 비슷한 상품 많은데 이게 뭐가 다르지? |
| **벽 넘기 전략** | 4종 부위 비주얼 그리드 + 골라담기 인터랙션 강조. "마트에선 불가능한, 4종 부위 자유 선택" USP 부각 |
| **핵심 심리 원칙** | 자율성 편향(Autonomy Bias) — 선택권이 주어지면 통제감을 느끼고 만족도가 높아짐. "내가 직접 고른다"는 재미 |

### Stage 4: 의심 (Doubt)

| 항목 | 내용 |
|------|------|
| **고객 내면 대화** | "근데 이렇게 싼 데는 이유가 있지 않나? 진짜 국내산 맞아? 양은 얼마나 되는 거야? 맛은 괜찮으려나..." |
| **감정 상태** | 관심 → 경계 + 합리적 의심 |
| **넘겨야 할 벽** | **신뢰의 벽** — 너무 좋은 조건 = 뭔가 함정이 있을 것 |
| **벽 넘기 전략** | 실제 리뷰(평점 5.0), 국내산 한돈 인증, 냉장 배송 과정 공개, 교환/반품 정책 명시. 의심을 회피하지 않고 정면 돌파 |
| **핵심 심리 원칙** | 사회적 증거(Social Proof) + 투명성 — 다른 구매자의 만족 후기와 구체적 정보 공개가 의심을 녹임 |

### Stage 5: 욕구 (Desire)

| 항목 | 내용 |
|------|------|
| **고객 내면 대화** | "아 진짜 맛있겠다... 주말에 가족이랑 삼겹살에 소주 한잔 하면 딱인데. 캠핑 갈 때도 가져가면 좋겠다" |
| **감정 상태** | 경계 해제 → 감각적 욕구 폭발 |
| **넘겨야 할 벽** | **욕구 전환의 벽** — "좋긴 한데..."에서 "진짜 갖고 싶다"로의 전환 |
| **벽 넘기 전략** | 시즐컷(굽는 소리, 육즙, 연기) + 먹방 장면 + "받자마자 바로 구워 드세요" 즉시 만족 시나리오 제시 |
| **핵심 심리 원칙** | 감각 전이(Sensory Transfer) — 시각적 자극이 미각/후각을 활성화시켜 "먹고 싶다"는 본능적 욕구 발동 |

### Stage 6: 망설임 (Hesitation)

| 항목 | 내용 |
|------|------|
| **고객 내면 대화** | "사고 싶긴 한데... 배송비가 3,500원이네. 다음에 살까? 좀 더 찾아볼까?" |
| **감정 상태** | 욕구 → 합리화 시도 + 이탈 충동 |
| **넘겨야 할 벽** | **결정의 벽** — 구매를 미루게 만드는 모든 작은 저항들 |
| **벽 넘기 전략** | "지금 이 가격이 언제까지인지 모릅니다" 희소성 + "5팩 담으면 무료배송 + 냉장고에 항상 고기가" 묶음 혜택 + 손실 프레이밍 |
| **핵심 심리 원칙** | 손실 회피(Loss Aversion) — "이 가격을 놓치면 다시 정가로 돌아갑니다"가 "사지 않는 것 = 손해"로 전환 |

### Stage 7: 불안해소 (Reassurance)

| 항목 | 내용 |
|------|------|
| **고객 내면 대화** | "혹시 맛없으면? 상태 안 좋으면 어쩌지? ...근데 반품도 되고, 불량이면 배송비도 안 내도 되네" |
| **감정 상태** | 망설임 → 안심 → 결제 확신 |
| **넘겨야 할 벽** | **후회 공포의 벽** — 결제 버튼 누르기 직전의 마지막 불안 |
| **벽 넘기 전략** | 교환/반품 7일 이내 보장, 불량 시 배송비 쇼핑몰 부담 명시, "만족 못 하시면 100% 교환" 무위험 메시지 |
| **핵심 심리 원칙** | 제로 리스크 편향(Zero-Risk Bias) — "위험 부담 제로"를 인식하면 마지막 심리 장벽이 사라짐 |

## 2. 6개 ZONE별 심리 트리거 배치 맵

---

### ZONE 1: 시선 강탈 (Attention Hijack)

| 항목 | 내용 |
|------|------|
| **목적** | 무한 스크롤 속에서 1.5초 안에 시선을 멈추게 한다 |
| **핵심 심리 원칙** | **인지 부조화** + **앵커링** |
| **실행안** | |

**헤드라인 구조:**
```
[태그] 국내산 한돈
[메인 카피] 9,500원에 골라담는 냉장 삼겹살
[서브] 정가 15,500원 → 39% 파격 할인 중
```

**심리 트리거 배치:**
- **앵커링**: 정가 15,500원 취소선 → 9,500원 빨간색 강조 → "6,000원 절약"
- **인지 부조화**: "국내산 한돈" + "9,500원"의 조합이 기존 가격 인식과 충돌
- **시각 자극**: 시즐컷 이미지 (지글거리는 삼겹살) 배경
- **숫자의 힘**: "39% OFF" 뱃지, "4종" 부위 수 강조

**비주얼 요소:**
- 풀 와이드 히어로 이미지: 숯불 위 삼겹살 시즐컷
- 좌측 상단: "국내산 한돈" 뱃지 (그린 + 골드)
- 중앙: 가격 앵커링 블록 (취소선 + 할인가 + 할인율)
- 우측 또는 하단: 4종 부위 미니 썸네일 그리드

---

### ZONE 2: 공감 + 관심 유발 (Empathy & Interest)

| 항목 | 내용 |
|------|------|
| **목적** | "이건 바로 나를 위한 상품"이라는 관련성을 인식시킨다 |
| **핵심 심리 원칙** | **미러링** + **자율성 편향** + **프레이밍 효과** |
| **실행안** | |

**공감 카피 구조:**
```
[고민 터치] "마트 삼겹살은 비싸고, 수입산은 불안하고..."
[해결 제시] "국내산 한돈 냉장육을 이 가격에, 부위까지 골라담으세요"
[차별화] "삼겹살 · 오겹살 · 목살 · 항정살 — 4종 자유 선택"
```

**심리 트리거 배치:**
- **미러링**: 타겟 페르소나의 일상 고민을 정확히 반영 → "이건 나 얘기잖아"
- **자율성 편향**: "골라담기" 키워드 반복 → 선택하는 재미와 통제감 부여
- **프레이밍**: "비싼 국내산 vs 불안한 수입산"이라는 이분법에서 → "제3의 선택지" 포지셔닝
- **비교 대비**: 경쟁 대비표 간소화 버전 (마트가 / 온라인가 / 본 상품가)

**비주얼 요소:**
- 4종 부위 고퀄리티 그리드 (삼겹살 / 오겹살 / 목살 / 항정살)
- 각 부위별 한 줄 설명 (예: "삼겹살 — 캠핑의 꽃, 소주의 단짝")
- 구이용/수육용 선택 아이콘 표시
- "골라담기" 인터랙션 느낌의 레이아웃 (체크박스/카드 스타일)

---

### ZONE 3: 욕구 폭발 (Desire Explosion)

| 항목 | 내용 |
|------|------|
| **목적** | 이성적 관심을 본능적 욕구로 전환한다 — "먹고 싶다"를 참을 수 없게 |
| **핵심 심리 원칙** | **감각 전이** + **심적 시뮬레이션** + **소유 효과(Endowment Effect)** |
| **실행안** | |

**욕구 자극 카피 구조:**
```
[감각 자극] "두툼한 삼겹살이 숯불 위에서 지글지글"
[시나리오] "퇴근 후 30분, 가족 모두가 '오늘 저녁 최고!'라고 할 식탁"
[소유 시뮬레이션] "냉장고를 열면 언제든 꺼내 구울 수 있는 국내산 한돈"
```

**심리 트리거 배치:**
- **감각 전이**: 지글거리는 소리, 피어오르는 연기, 탱글탱글한 육질 묘사 → 침샘 자극
- **심적 시뮬레이션**: "주말 가족 삼겹살 파티", "캠핑장에서 친구들과 고기 굽기" 등 구체적 사용 장면
- **소유 효과**: "이미 내 냉장고에 있다면?" → 소유한 상태를 상상하게 해서 포기하기 어렵게 만듦
- **현재 편향**: "받자마자 바로 구워 드세요" — 냉장이라 해동 불필요 → 즉시 만족 약속

**비주얼 요소:**
- 시즐컷 대형 이미지 (연기, 육즙, 기름 방울이 보이는 동적 컷)
- 라이프스타일 컷 2~3장 (가족 식탁, 캠핑, 친구 모임)
- 조리 과정 스텝 이미지 (포장 → 꺼냄 → 그릴 → 완성)
- 마블링 클로즈업 컷 (육질 품질 시각 증거)

---

### ZONE 4: 신뢰 구축 (Trust Building)

| 항목 | 내용 |
|------|------|
| **목적** | "사도 괜찮겠다"는 합리적 확신을 심어준다 |
| **핵심 심리 원칙** | **사회적 증거** + **권위** + **투명성 효과** |
| **실행안** | |

**신뢰 구축 카피 구조:**
```
[사회적 증거] "구매 고객 평점 5.0 — '이 가격에 이 맛은 사기급'"
[권위/인증] "국내산 한돈 인증 — 산지에서 냉장 직배송"
[투명성] "원산지: 국내산 | 상태: 냉장 | 등급: ___"
```

**심리 트리거 배치:**
- **사회적 증거**: 리뷰 평점 5.0 대형 표시 + 실제 리뷰 발췌 (구체적이고 생생한 후기)
- **권위**: "국내산 한돈" 인증 마크/뱃지, 원산지 표시 의무 정보 강조
- **투명성**: 포장 과정, 냉장 배송 과정, 원산지 정보를 숨기지 않고 공개
- **구체적 숫자**: 리뷰 5건 (수치화), 평점 5.0/5.0, 배송 1~3일 등 구체적 수치

**비주얼 요소:**
- 리뷰 카드 UI (평점 별점 + 실제 후기 텍스트 + 구매자 표시)
- 인증 뱃지 배너 (국내산 한돈 마크)
- 냉장 배송 과정 인포그래픽 (산지 → 가공 → 냉장 포장 → 배송 → 식탁)
- 신뢰 아이콘 모음 (국내산, 냉장, HACCP 등)

---

### ZONE 5: 긴급성 + 행동 촉발 (Urgency Trigger)

| 항목 | 내용 |
|------|------|
| **목적** | "나중에"를 "지금 당장"으로 전환한다 |
| **핵심 심리 원칙** | **희소성** + **손실 회피** + **현상유지 편향 파괴** |
| **실행안** | |

**긴급성 카피 구조:**
```
[희소성] "이 가격은 한정 수량 소진 시 종료됩니다"
[손실 프레이밍] "지금 안 담으면, 다음엔 정가 15,500원"
[기회비용] "이 가격에 국내산 한돈 냉장육을 만나는 건 흔치 않습니다"
```

**심리 트리거 배치:**
- **희소성**: "한정 물량", "재입고 미정" 등 수량 제한 암시
- **손실 회피**: "39% 할인은 지금뿐" → 사지 않는 것 = 6,000원 손해로 프레이밍
- **현상유지 편향 파괴**: "오늘도 비싼 수입 냉동육에 만족하시겠습니까?" → 현재 상태의 불만족 환기
- **시간 압박**: "할인 종료 임박" 뉘앙스 (과도한 카운트다운은 지양, 자연스러운 긴급감)

**비주얼 요소:**
- 할인 종료 알림 배너 (부드러운 긴급감)
- 정가 → 할인가 비교 재강조 (Zone 1 앵커링 리마인드)
- "지금 담기" CTA 버튼 (식욕 레드 #E63312)
- "5만원 이상 무료배송까지 OO원 남았어요" 프로그레스 바

---

### ZONE 6: 구매 전환 + 안심 (Conversion & Reassurance)

| 항목 | 내용 |
|------|------|
| **목적** | 마지막 불안을 제거하고 결제 버튼을 누르게 한다 |
| **핵심 심리 원칙** | **제로 리스크 편향** + **일관성 원칙** + **치알디니의 호혜성** |
| **실행안** | |

**전환 카피 구조:**
```
[불안 해소] "만족 못 하시면 7일 이내 교환/반품 OK"
[무위험] "불량 시 배송비도 저희가 부담합니다"
[최종 CTA] "국내산 한돈 4종 골라담기 → 지금 주문하기"
[보너스] "50,000원 이상 주문 시 배송비 무료!"
```

**심리 트리거 배치:**
- **제로 리스크**: 교환/반품 정책 명확히 표시 → "잘못 사도 괜찮아" 안심
- **일관성**: "여기까지 읽으셨다면, 이미 마음은 정해지셨죠?" → 이미 투자한 시간과 관심에 일관되게 행동하도록 유도
- **호혜성**: "불량 시 배송비 쇼핑몰 부담" = 판매자가 먼저 위험을 감수 → 고객은 호의에 보답하려는 심리
- **묶음 유도**: "5만원 이상 무료배송" 안내 → "어차피 살 거 더 담자" 추가 구매 촉진

**비주얼 요소:**
- 대형 CTA 버튼 (식욕 레드, 라운드, "지금 골라담기")
- 안심 배지 3종 (교환보장 / 냉장배송 / 국내산)
- 무료배송 달성 프로그레스 바
- 간단한 주문 프로세스 안내 (3단계: 부위선택 → 용도선택 → 결제)

## 3. 감정 곡선 설계

### 전체 감정 흐름

```
감정 강도
높음 │                          ★ 욕구 폭발
     │                        ╱    ╲
     │                      ╱        ★ 긴급!
     │         ★ 희망      ╱          ╲
     │        ╱   ╲      ╱              ★ 안심+확신
     │      ╱       ╲  ╱
     │    ╱          ╲╱ (의심 → 해소)
     │  ╱
     │╱ ★ 부정(고민)
낮음 │───────────────────────────────────────→ 스크롤
     Zone1    Zone2    Zone3   Zone4   Zone5  Zone6
     시선멈춤  공감+관심  욕구자극  신뢰구축  긴급성   전환
```

### 단계별 감정 설계

| 단계 | 감정 | 강도 | 설계 의도 |
|------|------|------|-----------|
| **Zone 1** | 놀라움 + 호기심 | ★★★☆☆ | 무심한 스크롤에서 "어?" 하고 멈추는 순간. 가격 충격으로 인지 부조화 유발 |
| **Zone 2 전반** | 부정적 공감 | ★★☆☆☆ | "맞아, 나도 고기값 때문에 스트레스야..." 현재 고민과 불만을 건드림. 감정의 저점 |
| **Zone 2 후반** | 희망 + 기대 | ★★★☆☆ | "근데 이게 된다고?" 해결책 제시로 부정 → 긍정 전환. 4종 골라담기의 재미 |
| **Zone 3** | 강렬한 욕구 | ★★★★★ | "먹고 싶다...!" 감각 자극 최대치. 시즐컷으로 본능적 식욕 자극. 감정 곡선의 최고점 |
| **Zone 4 초반** | 의심 + 경계 | ★★★☆☆ | "너무 좋은데... 함정 아니야?" 의도적 감정 하락. 이성적 점검 시간 허용 |
| **Zone 4 후반** | 안심 + 해소 | ★★★★☆ | "아, 리뷰도 좋고 인증도 있네. 괜찮겠다" 의심 해소 후 더 강한 확신으로 리바운드 |
| **Zone 5** | 긴급함 + 흥분 | ★★★★★ | "지금 안 사면 손해다!" 손실 회피 발동. 행동 에너지 폭발 |
| **Zone 6** | 안심 + 만족감 | ★★★★☆ | "잘 샀다, 기대된다" 결제 후 후회 없는 만족. 반품 보장으로 잔여 불안 제거 |

### 감정 곡선의 핵심 설계 원칙

1. **부정 → 긍정 전환 (Zone 2)**: 먼저 고객의 아픔을 인정해야 해결책이 빛남. 부정적 감정 없이 바로 장점만 나열하면 "광고 같다"고 느낌
2. **욕구 피크 후 의도적 하락 (Zone 3→4)**: 감정이 최고점에서 바로 CTA로 가면 "충동구매 유도" 느낌. 의심 단계를 자연스럽게 배치해 이성적 만족도 제공
3. **V자 리바운드 (Zone 4→5)**: 의심을 해소한 후의 확신은 의심 전보다 더 강함. 이 반등 에너지를 긴급성과 결합해 구매 행동으로 전환
4. **안심 착지 (Zone 6)**: 긴급성으로 올린 흥분을 부드럽게 안심으로 전환. "잘 샀다"는 감정으로 마무리해야 재구매 + 긍정 리뷰로 연결

## 부록: 심리 원칙 총정리 맵

| 심리 원칙 | 적용 위치 | 적용 방법 |
|-----------|-----------|-----------|
| **앵커링 (Anchoring)** | Zone 1, 5 | 정가 15,500원 취소선 → 9,500원 인지, 마트 가격 대비 절약 강조 |
| **인지 부조화 (Cognitive Dissonance)** | Zone 1 | "국내산 한돈 = 비싸다" 기존 믿음과 충돌하는 9,500원 |
| **미러링 (Mirroring)** | Zone 2 | 타겟의 일상 고민을 정확히 반영하여 공감대 형성 |
| **자율성 편향 (Autonomy Bias)** | Zone 2 | 4종 부위 + 용도 선택권 부여 → 통제감과 만족감 |
| **프레이밍 효과 (Framing)** | Zone 2, 5 | "비싼 국내산 vs 불안한 수입산" 이분법에서 제3의 선택지 제시 |
| **감각 전이 (Sensory Transfer)** | Zone 3 | 시즐컷으로 시각 → 미각/후각 연결, 식욕 자극 |
| **심적 시뮬레이션 (Mental Simulation)** | Zone 3 | 구체적 사용 장면 묘사로 "이미 먹고 있는" 상상 유도 |
| **소유 효과 (Endowment Effect)** | Zone 3 | "냉장고에 항상 있다면?" 소유 상태 상상 → 포기 어려움 |
| **사회적 증거 (Social Proof)** | Zone 4 | 평점 5.0, 실제 후기, 구매자 수 |
| **권위 (Authority)** | Zone 4 | 국내산 한돈 인증, 원산지 표시, 생산 과정 |
| **투명성 효과** | Zone 4 | 정보 숨기지 않고 공개 → 신뢰 형성 |
| **희소성 (Scarcity)** | Zone 5 | 한정 수량, 할인 종료 임박, 재입고 미정 |
| **손실 회피 (Loss Aversion)** | Zone 5 | "지금 안 사면 정가로 돌아감" = 6,000원 손해 |
| **제로 리스크 편향 (Zero-Risk Bias)** | Zone 6 | 7일 교환/반품, 불량 시 배송비 쇼핑몰 부담 |
| **일관성 (Consistency)** | Zone 6 | 여기까지의 관심 투자에 일관되게 행동하도록 유도 |
| **호혜성 (Reciprocity)** | Zone 6 | 판매자가 먼저 위험 부담 → 고객의 보답 심리 |
| **현재 편향 (Present Bias)** | Zone 3, 5 | 냉장이라 해동 없이 바로 조리 → 즉시 만족 |

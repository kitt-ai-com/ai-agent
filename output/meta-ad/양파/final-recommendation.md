# 최종 광고 소재 추천 - 국내산 양파

## Executive Summary

**제품**: 국내산 양파 실중량 7kg 이상, 8,900원 무료배송
**목표**: ROAS 3.5 이상, 월 매출 380만원
**예산**: 월 100만원 (일 33,000원 → 50,000원 점진적 증액)
**기간**: 4주
**핵심 전략**: 마트 대비 가격 경쟁력 + 단단함·당도 선별 품질 + 무료배송 편의성
**차별점**: kg당 1,271원 (마트 대비 50% 저렴) + 무료배송 + 단단함·당도 선별

## 1차 테스트 소재 (최우선 실행)

### 추천 소재 TOP 5

#### 1순위: V1-1 (마트 비교형) - **최우선 추천**
- **제목**: 마트 가지 마세요
- **문구**: 마트 양파 1kg 2,500원? 여기는 7kg에 8,900원! kg당 1,271원, 무료배송까지. 무거운 양파 이제 집으로 받으세요!
- **이미지**: 정1 (바구니 양파 + 7kg 꽉 채워)
- **타겟**: 30-55세 주부 (마트 이용 고객)
- **예산 배분**: 30%
- **예상 ROAS**: 3.8
- **이유**: 가격 비교 직관적, 마트 대비 강점 명확, 무료배송 편의성
- **게재 시간**: 오전 9-12시 집중

#### 2순위: V2-2 (요리 맛 향상 감성형)
- **제목**: 양파 바꿨더니 요리가 달라졌어요
- **문구**: (롱카피) 남편이 "오늘 카레 왜 이렇게 맛있어?"... 비결은 단단하고 달콤한 국내산 양파...
- **이미지**: 정2 (양파부터 다르면 요리가 달라집니다)
- **타겟**: 30-50세 요리하는 가정
- **예산 배분**: 25%
- **예상 ROAS**: 3.9
- **이유**: 감성적 공감대, 요리 활용도 강조, 높은 전환율
- **게재 위치**: 피드 중심

#### 3순위: V2-1 (주부 장보기 공감형)
- **제목**: 마트 양파, 이제 그만
- **이미지**: 정1 또는 정2
- **타겟**: 30-55세 주부
- **예산 배분**: 20%
- **예상 ROAS**: 3.7
- **이유**: 장보기 불편함 공감, 스토리텔링으로 높은 전환율
- **게재 위치**: 피드

#### 4순위: V1-2 (품질 선별형)
- **제목**: 물렁한 양파 NO
- **문구**: 단단함·당도 기준 통과한 것만 선별! 물렁한 건 빼고 단단한 국내산 양파만 7kg 꽉 채워 드립니다. 8,900원 무료배송!
- **이미지**: 정4 (물렁한 건 빼고 단단한 것만 선별)
- **타겟**: 품질 중시 고객
- **예산 배분**: 15%
- **예상 ROAS**: 3.9
- **이유**: 품질 차별화, 물렁한 양파 불만 해결

#### 5순위: V2-7 (리뷰 종합형)
- **제목**: 고객님들이 증명합니다
- **이미지**: 정1 또는 정4
- **타겟**: 리뷰 중시 고객
- **예산 배분**: 10%
- **예상 ROAS**: 4.0
- **이유**: 사회적 증거로 신뢰도 구축

## 문제 해결 가이드

### 문제 1: 전환이 없어요 (Day 1-2)
→ **정상입니다.** 학습 기간 (24-48시간) 필요. Day 3부터 데이터 분석.

### 문제 2: CTR이 낮아요 (< 2.0%)
→ V1-1 "마트 가지 마세요"로 교체, 정1 이미지 (7kg 꽉 채워) 강화

### 문제 3: CPC가 너무 높아요 (> 250원)
→ 입찰 상한 220원 설정, Audience Network 제외, 주부 타겟 집중

### 문제 4: ROAS가 목표에 못 미쳐요
→ 리타겟팅 비중 증가, 저성과 타겟 중단, 주부 타겟 집중 투자, 2박스 구매 유도

## 성공 팁

1. **"마트 대비" 메시지**: 모든 소재에서 마트 비교 필수. 양파의 핵심 구매 채널이 마트이므로 이것이 최대 차별점.
2. **"무료배송" 강조**: 양파는 무거워서 배송비가 중요한 구매 요소. 무료배송이 강력한 세일즈 포인트.
3. **주부 타겟 집중**: 45% 이상 예산. 양파의 주 구매자.
4. **저녁 시간대 집중**: 저녁 18-22시에 40% 예산. 내일 요리 준비 동기.
5. **이미지 로테이션**: 정1~정4를 주기적으로 교체하여 피로도 방지.
6. **봄 양파 시즌 활용**: 3~4월 봄 양파 시즌 긴급성 메시지.

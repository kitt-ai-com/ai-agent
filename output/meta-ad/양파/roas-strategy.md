# ROAS 최적화 전략 - 국내산 양파

## 목표
- **판매 전환 극대화**

- **ROAS 3.5 이상 달성** (가성비 + 무료배송 차별점 활용)

## 현재 상황
- **제품**: 국내산 양파 실중량 7kg 이상
- **가격**: 8,900원 (할인 없음)
- **배송비**: 무료배송
- **목표 ROAS**: 3.5 이상
- **타겟**: 30-55세 주부, 25-45세 맞벌이, 40-65세 건강 중시층
- **예산**: 월 100만원 (일 33,000원 가정)
- **제외 지역**: 제주 및 도서산간

## 문제 해결 시나리오

### 시나리오 1: CTR이 2.0% 미만
**해결책**:
1. 소재를 V1-1 (마트 비교형)으로 교체 - 가장 직관적
2. 이미지를 정1 (7kg 꽉 채워)로 변경 - 가격 눈에 띄게
3. 타겟 연령대 조정 (30-50세 여성 집중)

### 시나리오 2: 전환율이 2.5% 미만
**해결책**:
1. 랜딩 페이지 로딩 속도 확인
2. 제품 상세 페이지 "8,900원 무료배송" 강조 확인
3. 타겟팅 재설정 (식품 구매 이력 고객)
4. 리타겟팅 비중 증가

### 시나리오 3: ROAS가 3.0 미만
**해결책**:
1. 입찰 상한선 하향 조정 (220원 → 180원)
2. Audience Network 제외
3. 고ROAS 세그먼트에만 집중 (주부 30-55세)
4. 리타겟팅 비중 증가 (전환율 높음)
5. 2박스 구매 유도 (AOV 상승)

### 시나리오 4: 광고 피로도 (Frequency 3.0 이상)
**해결책**:
1. 이미지 변경 (정1↔정2↔정3↔정4 로테이션)
2. 문구 변형 (같은 메시지 다른 표현)
3. 타겟 확대 (Lookalike 비율 상향)
4. 게재 빈도 상한 설정 (일 2회)

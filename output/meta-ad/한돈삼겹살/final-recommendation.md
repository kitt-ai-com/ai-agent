## 🔧 문제 해결 가이드

### 문제 1: 전환이 없어요 (Day 1-2)
→ **정상입니다.** 학습 기간 (24-48시간) 필요. Day 3부터 데이터 분석.

### 문제 2: CTR이 낮아요 (< 2.0%)
→ 소재 교체 (V1-3 소가구형 또는 V2-1 외식 대체형으로)
→ 할인율(36%) 더 크게 표시
→ 타겟 연령대 조정 (25-35세 집중)

### 문제 3: CPC가 너무 높아요 (> 300원)
→ 입찰 상한 설정 (250원)
→ Audience Network 제외
→ 타겟 확대 (Lookalike)

### 문제 4: ROAS가 목표에 못 미쳐요 (< 3.0)
→ 1-2인 가구 타겟 비중 증가 (최고 전환율)

→ 저성과 광고 세트 중단
→ 고ROAS 세트 집중 투자

### 문제 5: 예산이 너무 빨리 소진돼요
→ 일 예산 상한 확인
→ 입찰 전략 조정 (최저 단가 → 비용 상한)

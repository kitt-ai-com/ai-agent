## 🚨 문제 해결 시나리오

### 시나리오 1: CTR이 2.0% 미만

**원인 분석**:
- 소재가 타겟에게 매력적이지 않음
- 크리에이티브 품질 낮음
- 타겟팅 부적절

**해결책**:
1. 소재 즉시 교체 (V1-6 한정 수량형, V2-6 리뷰형으로)
2. 이미지/비디오 품질 개선 (고화질, 식욕 자극)
3. 타겟 연령대 조정 (±5세)
4. 가격/할인율 더 크게 표시

---

### 시나리오 2: 전환율이 2.5% 미만

**원인 분석**:
- 랜딩 페이지 문제
- 가격 경쟁력 부족
- 타겟 미스매치

**해결책**:
1. 랜딩 페이지 최적화 (로딩 속도, 모바일 최적화)
2. 추가 할인 쿠폰 제공 (5% 추가 할인)
3. 타겟팅 재설정 (1-2인 가구 집중)
4. 신선도 강조 (냉장육 장점)

---

### 시나리오 3: ROAS가 3.0 미만

**원인 분석**:
- CPA 너무 높음
- 저가치 트래픽 유입
- 입찰 전략 비효율

**해결책**:
1. 입찰 상한선 하향 조정 (250원 → 200원)
2. Audience Network 제외
3. 고ROAS 세그먼트에만 집중 투자
4. 리타겟팅 비중 증가 (전환율 높음)
5. 1-2인 가구 타겟 집중 (전환율 높음)

---

### 시나리오 4: 광고 피로도 (Frequency 2.5 이상)

**원인 분석**:
- 동일 타겟에게 과도한 노출
- 타겟 풀 너무 좁음

**해결책**:
1. 크리에이티브 refresh (새로운 이미지/문구)
2. 타겟 확대 (Lookalike 비율 상향)
3. 게재 빈도 상한 설정 (일 2회)
4. 신규 소재 추가 (계절별, 시즌별)

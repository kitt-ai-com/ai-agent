# 최종 광고 소재 추천 - 한우곰탕

## 📋 Executive Summary

**제품**: 가마솥에 24시간 직접끓인 한우곰탕 500g × 10팩, 14,000원
**목표**: ROAS 3.5 이상, 월 매출 650만원
**예산**: 월 168만원 (일 40,000원 → 70,000원 점진적 증액)
**기간**: 4주
**핵심 전략**: 5분 간편성 최우선 + 타겟 세분화 + 배송비 리프레이밍
**차별점**: 한우우족 대비 압도적 간편성 (2시간 vs 5분)

## 🔧 문제 해결 가이드

### 문제 1: 전환이 없어요 (Day 1-2)
→ **정상입니다.** 학습 기간 (24-48시간) 필요. Day 3부터 데이터 분석.

### 문제 2: CTR이 낮아요 (< 2.0%)
→ V1-1 "5분 완성형"으로 교체, 타이머 이미지 강화, 아침 시간대 집중

### 문제 3: 배송비 때문에 이탈이 많아요
→ 메시지 리프레이밍:
- "총 18,000원 ÷ 10팩 = 1,800원 vs 식당 8,000원"
- "배송비 포함해도 60% 저렴"
- 2세트 구매 유도 (배송비 분산)

### 문제 4: CPC가 너무 높아요 (> 300원)
→ 입찰 상한 250원 설정, Audience Network 제외, 타겟 좁히기 (직장인 집중)

### 문제 5: ROAS가 목표에 못 미쳐요
→ 리타겟팅 비중 증가 (장바구니 이탈 20% → 30%), 저성과 타겟 중단, 직장인 타겟 집중 투자

## 💡 성공 팁

1. **5분 강조**: 모든 소재에서 "5분" 키워드 필수. 이것이 최대 차별점.
2. **아침 시간대 집중**: 직장인 타겟은 아침 6-9시 예산 35% 집중.
3. **배송비 리프레이밍**: "1팩당 1,800원 (배송비 포함)" 메시지로 부담 완화.
4. **리타겟팅 중요**: 전환율 가장 높음 (10%+), 반드시 활용.
5. **크리에이티브 피로도**: 2주마다 이미지/문구 교체.
6. **타겟 집중**: 직장인 + 1인 가구 75% 예산 집중.

## 💡 프로젝트 핵심 요약

### 제안 금액
- **152,000,000원 (VAT 별도)**
- VAT 포함: 167,200,000원

### 개발 기간
- **4개월 (16주)**

### 개발 범위
- **Phase 1-3**: 기반 구축 + 핵심 API + 웹 대시보드
- AI 채팅 제외 (Phase 4+ 별도)

### 핵심 기능
1. 주문 엑셀 자동 업로드 (쿠팡/스마트스토어 등)
2. 공급사별 발주서 자동 생성
3. 송장 자동 매칭 (공급사 송장 ↔ 주문)
4. 등급별 차등 가격 + 개별 설정
5. 정산 자동 계산 (수수료, 반품/환불)

## 📊 기대 효과

### 정량적 효과
- 주문 처리 시간 **80% 단축** (4시간 → 50분)
- 발주 오류율 **90% 감소** (5% → 0.5%)
- 인건비 **연 1,920만원 절감**
- 총 절감액 **연 3,800만원**

### 투자 회수
- **약 7.7년** (순수 절감 기준)
- 실제는 더 짧을 가능성 (플랫폼 수수료 수익, 거래량 증가)

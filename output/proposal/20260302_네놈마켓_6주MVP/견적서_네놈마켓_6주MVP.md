## 1. 프로젝트 요약

네놈마켓 B2B 유통 플랫폼의 미구현 핵심 기능을 6주 내에 완성하여, **실제 운영이 가능한 MVP**를 구축합니다.

| 현황 | 수치 |
|------|------|
| 전체 기능 | 68개 |
| 현재 구현 완료 | 29개 (42.6%) |
| 본 프로젝트 개발 대상 | 24개 |
| MVP 완성 후 진행률 | **53개 (77.9%)** |

**개발 대상 핵심 영역:**

| 영역 | 기능 수 | 주요 내용 |
|------|:------:|----------|
| 정산/매출 관리 | 6 | 매출 집계, 수수료 자동 계산, 클레임 차감, 엑셀 다운로드 |
| 송장 자동 매칭 | 2 | 공급사 송장 엑셀 업로드 시 주문별 자동 매칭 |
| CS 관리 | 5 | 문의 게시판, 클레임(반품/환불/교환) 접수~처리~정산 연동 |
| 셀러 통계/정산 | 5 | 매출 통계, 판매량, 주문 추이, 정산 내역 조회 |
| 알림/공지 | 2 | 공지사항, 가격 변동/품절 인앱 알림 |
| 기반 보완 | 2 | 재고 부족 알림, 발주 양식 템플릿 |
| 통합 QA | — | 기능 테스트, 회귀 테스트, 버그 수정 |
| **합계** | **24** | |

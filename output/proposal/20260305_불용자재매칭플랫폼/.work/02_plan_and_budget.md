### 2.3 역할별 책임 매트릭스 (RACI)

| 태스크 | PM | 디자이너 | 프론트엔드 | 백엔드 | QA |
|--------|----|---------|-----------|--------|----|
| 요구사항 분석 | **R** | C | C | C | I |
| UI/UX 설계 | A | **R** | C | I | I |
| API 설계 | A | I | C | **R** | I |
| 프론트 개발 | A | C | **R** | C | I |
| 백엔드 개발 | A | I | C | **R** | I |
| AI 매칭 개발 | A | I | I | **R** | I |
| 테스트 | A | I | C | C | **R** |
| 배포 | A | I | C | **R** | I |

- **R**: Responsible (실행 책임)
- **A**: Accountable (최종 승인)
- **C**: Consulted (협의 대상)
- **I**: Informed (정보 공유 대상)

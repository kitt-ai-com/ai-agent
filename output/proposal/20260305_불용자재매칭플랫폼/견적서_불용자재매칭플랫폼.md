## 3. 간소화 MVP 범위

### 3.1 포함 범위 (핵심 5개 화면)

✅ **핵심 기능만 선별**

| 화면 ID | 화면명 | 주요 기능 | 비고 |
|---------|--------|----------|------|
| S-001 | 회원가입/로그인 | OCR 사업자등록증 인식, 이메일 로그인 | 통합 화면 |
| S-101 | 자재 등록 (엑셀) | 엑셀 파일 업로드, 템플릿 다운로드, 검증 | 판매자 핵심 |
| S-201 | AI 매칭 결과 | 추천 자재 목록, 매칭 점수 | 구매자 핵심 |
| S-202 | 자재 상세 | 자재 정보, 거래 요청 버튼 | |
| S-301 | 거래 관리 (간소화) | 거래 요청/승인, 연락처 공유 | 최소 기능 |

**핵심 기능**
- ✅ 회원가입/로그인 (OCR 포함)
- ✅ 자재 엑셀 일괄 업로드 (템플릿 제공)
- ✅ AI 자동 매칭 (OpenAI Embedding, 기본 알고리즘)
- ✅ 거래 요청/승인 (간소화)
- ✅ 기본 대시보드 (My Page 통합)

**기술 스택**
- Next.js 14 + Supabase + OpenAI API + CLOVA OCR
- 기본 템플릿: Shadcn/ui (커스텀 디자인 최소화)
- Vercel 무료 플랜 + Supabase 무료 플랜

### 3.2 제외/간소화 범위

❌ **제외 기능**
- ❌ 소셜 로그인 (이메일 로그인만)
- ❌ 실시간 알림 (이메일 알림만)
- ❌ 고급 필터링
- ❌ 자재 수정/삭제 기능 (관리자만 가능)
- ❌ 상세 통계/리포트

🔻 **간소화 기능**
- 🔻 UI 디자인: 기본 템플릿 사용 (커스텀 디자인 최소)
- 🔻 대시보드: 간단한 목록 형태
- 🔻 거래 관리: 최소 워크플로우만
- 🔻 AI 매칭: 기본 알고리즘 (고도화 없음)

- 모바일 앱

### 9.2 추가 비용 발생 시

다음 항목은 별도 견적이 필요합니다:

- 추가 화면 개발: 화면당 50~100만원
- 고급 기능 추가: 기능별 100~200만원
- 모바일 앱: 1,000만원 이상

### 9.3 견적 유효기간

- 본 견적은 **제안일로부터 30일간** 유효합니다
- 30일 이후 계약 시 물가 변동을 고려하여 금액이 조정될 수 있습니다

### 9.4 기타 조건

- 계약금 입금 후 프로젝트가 시작됩니다
- 중도금은 2주차 회원/자재 기능 완료 시점에 청구됩니다
- 잔금은 4주차 프로덕션 배포 완료 및 검수 완료 시점에 청구됩니다

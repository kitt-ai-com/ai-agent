## 슬라이드 제작 가이드

### 견적서 디자인 원칙
1. **신뢰감**: 깔끔하고 전문적인 레이아웃
2. **명확성**: 비용 항목 투명하게 공개
3. **설득력**: ROI 명확하게 제시

### 색상 팔레트
- **배경**: 화이트 (#ffffff) 또는 연한 회색 (#f8f9fa)
- **주색상**: 네이비 블루 (#0f4c81)
- **강조색**: 주황색 (#ff6b35) - 금액
- **긍정**: 초록색 (#4ecca3) - 절감, 효과

### 폰트 사용
- **제목**: Pretendard Bold, 48pt
- **금액**: Pretendard ExtraBold, 60pt
- **본문**: Pretendard Regular, 20pt
- **작은 텍스트**: Pretendard Light, 16pt

### 레이아웃 구조

**슬라이드 1 (메인 견적서)**
- 상단: 제목 + 프로젝트명
- 중앙: 비용 테이블 (큰 비중)
- 하단: 지불 조건 플로우
- 최하단: 유지보수 정보

**슬라이드 2 (ROI)**
- 상단: 초기 투자 (박스)
- 좌측: 월간 절감 (차트)
- 우측: 매출 증대 (아이콘)
- 하단: ROI 요약 (큰 숫자)

### 시각 요소

**테이블 스타일**
```
┌────────────────────┐
│      헤더          │
├────────────────────┤
│  항목  │  금액     │
│  항목  │  금액     │
└────────────────────┘
```
- 헤더: 어두운 배경
- 교차 배경: 연한 회색/화이트

**플로우 차트**
```
┌──────┐    ┌──────┐    ┌──────┐
│계약금│ →  │중도금│ →  │잔 금│
└──────┘    └──────┘    └──────┘
```
- 박스: 라운드 코너
- 화살표: 굵게, 주황색

**ROI 차트**
```
월간 절감
█████████████ 1,800,000원

초기 투자
██████ 7,000,000원
```
- 막대 그래프
- 초록색 (절감) vs 파란색 (투자)

### 인쇄용 고려사항
- 여백: 상하좌우 충분히 (20mm 이상)
- 폰트 크기: 최소 16pt 이상
- 색상: CMYK 변환 대비

### 사용 시나리오

**시나리오 1: 간단 발표**
- 슬라이드 1만 사용
- 5분 내외 설명
- 빠른 의사결정 필요시

**시나리오 2: 상세 설명**
- 슬라이드 1 + 2 사용
- 10분 내외 설명
- ROI 강조 필요시

**시나리오 3: 인쇄용**
- 간단 1장 버전 사용
- PDF 출력
- 이메일 첨부용

## 💡 핵심 인사이트 3가지

### 1. 🟢 홍게 캠페인이 3배 더 효율적

```
CPA 비교
• 홍게: 20,246원
• 킹크랩: 60,402원 (3배 높음)

전환율 비교
• 홍게: 3.46%
• 킹크랩: 0.45% (7.7배 낮음)
```

**→ 홍게 캠페인을 중단한 것은 실수. 즉시 재개 필요.**

---

### 2. 🔴 킹크랩은 2,437명 클릭 → 11명만 구매

```
킹크랩 퍼널 분석
• 클릭: 2,437명
• 구매: 11명
• 이탈: 2,426명 (99.5%)
```

**문제점:**
- ❌ 가격이 너무 높거나
- ❌ 재고/옵션 부족이거나
- ❌ 랜딩페이지에 문제 있음

**→ 페이지 전면 점검 + 가격/프로모션 재검토 필요**

---

### 3. 🟢 킹크랩 광고 소재는 매우 우수함

```
킹크랩 트래픽 지표
• CTR: 6.03% (업계 평균의 3-6배)
• CPC: 273원 (업계 평균의 절반)
```

**→ 광고는 잘 만들어졌음. 문제는 전환 단계에 있음.**

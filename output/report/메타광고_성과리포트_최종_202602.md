## 💡 핵심 인사이트 3가지

### 1. 🔄 이전 분석과 정반대: 킹크랩이 실제론 더 효율적

**숨겨진 진실:**

```
CPA만 보면:
• 홍게: 20,246원 ✅
• 킹크랩: 60,402원 ❌

하지만 ROAS로 보면:
• 홍게: 1.10배 ❌ (손실 가능성)
• 킹크랩: 2.90배 ✅ (수익 2.9배)
```

**이유:**
- 킹크랩 객단가가 **7.8배** 높음 (175,173원 vs 22,297원)
- 고가 상품은 전환율이 낮아도 수익성이 더 좋을 수 있음

**→ CPA보다 ROAS가 더 중요한 지표!**

---

### 2. 🎯 킹크랩 전환율만 2배 올리면 월 200만원 추가 수익

**현재 상황:**
```
킹크랩 퍼널
• 클릭: 2,437명
• 구매: 11건 (0.45%)
• 매출: 192만원
```

**만약 전환율을 1%로 개선하면:**
```
• 구매: 24건 (2.2배)
• 추가 매출: 227만원
• 추가 광고비: 78만원 (동일 CPC 가정)
• 추가 순이익: 149만원 ✨
```

**개선 방법:**
- 랜딩페이지 최적화
- 리뷰/신뢰도 강화
- 가격 프로모션 (10-15% 할인도 수익 가능)

**→ 킹크랩 전환율 개선이 최우선 과제**

---

### 3. ⚠️ 홍게는 객단가 상승 없이는 적자 위험

**현재 홍게:**
```
• 객단가: 22,297원
• CPA: 20,246원
• 마진: 2,051원 (원가 제외 전)

원가 50% 가정 시:
• 실제 마진: -9,102원 (적자!)
```

**손익분기점:**
- ROAS 2.0배 이상 필요 (원가 50% 가정)
- 현재 1.1배는 위험 수준

**해결 방안:**
1. 객단가 상승 (크로스셀/업셀)
   - 2만원 → 3만원 목표 (ROAS 1.65배)
2. 또는 CPA 절감
   - 2만원 → 1.2만원 목표 (ROAS 1.85배)

**→ 홍게는 재개 전에 수익성 구조 개선 필요**

### 액션 2: 홍게는 수익성 개선 전까지 중단 유지

**현재 상태:** Inactive (중단됨)
**결론:** ✅ **중단이 옳은 판단**

**이유:**
- ROAS 1.1배는 원가 고려 시 적자 가능성 높음
- 재개보다는 구조 개선이 우선

**개선 방향:**

```
옵션 A: 객단가 상승 (추천)
• 홍게 + 소스 세트 상품 (2.5만 → 3.5만)
• 2개 세트 할인 (2.5만 → 4만)
• ROAS 1.1 → 1.5-1.8배로 개선

옵션 B: CPA 절감
• 타겟 최적화로 CPA 2만 → 1.2만
• ROAS 1.1 → 1.8배로 개선

옵션 C: 중단 유지 (현재)
• 킹크랩에 리소스 집중
```

**결정:** 홍게는 일단 중단 유지, 킹크랩 최적화 후 재검토
